# Web-Novel Foundations — 零基础网文写作速览

> 来源：番茄小说金番作者 **京祺**（10 年网文经验，代表作《野玫瑰》《我于深渊时见你》）
> 用途：story-studio 在**选题、构思、大纲、细纲、开头**阶段的底层方法论。适用所有品类。
> 定位：这是"心法"，与 `chapter-hooks`（钩子）、`climax-design`（高潮）、`title-design`（标题）互补。

---

## 一、如何写出好故事（构思心法）

### 1. 选题材 → 造场景 → 展开内容
先定一个大致题材，在题材里构造一个**具体场景**，再从场景展开。
> 例：重生题材 → 主角被恶毒女配一把火烧死 → 重生回到过去 → 故事启动。
先有"一个能立刻开写的场景"，比先有"宏大设定"更重要。

### 2. 找灵感：小人物牵出大事件
- 从社会热点/新闻事件提取**故事基调、案例原型、人物**。
- **四两拨千斤**：不一定从大事件切入，可用一个小人物/小事件牵出大背景，读者越看越上头。

### 3. 主角身份与特点：用"负资产"当钩子
主角必须"有点常人没有的东西"——但**不一定是超能力**，反向思考更好写、更抓人：
- 几个亿的负债 / 只剩三个月寿命 / 被灭门的凄惨身世 / 背负命案（须符合主流价值观：身不由己、被迫反击）。
- **突出特点可以是很糟糕的东西**。读者最爱"小人物逆风翻盘"——给他一个**天崩开局**，好写又抓人，最适合新手。

### 4. 目标要"活"，会随剧情变
给主角一个吸引人的目标，且目标**不是死的**，随剧情戏剧化而改变。
> 例：只剩三个月寿命 → 目标是"痛快花光钱不留遗憾" → 梭哈 30 万变 3000 万 → 钱花不完 → 主角**不想死了**，目标反转为"好好活着"。
目标的转折 = 故事吸引力的来源。

### 5. 冲突与对抗（最重要）
> 最能留住读者的，不是题材或噱头，而是**主角的社会身份 + 主角与配角之间的关系冲突/拉扯**。
同题材有人写得好看有人不好看，差别就在人物拉扯。
- **多线思维、反转、出其不意**，拒绝单线。
- 悬疑范例：主角目睹凶案本想报警 → 发现被杀者是虐待他、害死他母亲的继父 → 他选择见死不救 → 继父用一串钥匙求救、咽气 → 警察赶到认定主角是凶手（他还真有杀人动机）→ 真凶暗处盯着，警察是被凶手引来的 → **主角落入圈套**（局中局、套中套）。
- 精髓：设计**人物关系 + 藏在暗处的关系网**，让读者猜了又猜、意犹未尽。

### 6. 写作风格与标签（长期主义）
- 先练**自己的风格**，别盲目追榜单/热点（学会时热点已过）。做**创造热点**的人。
- 多读**经典**（经典网文/出版物/漫画/影视）：读一本经典十遍 > 读一百本流水网文。经典的框架、人设、起伏、台词都经过千锤百炼。
- 写作 = 炒菜：套路是步骤，**火候/佐料/下料顺序 = 你的灵性与天赋**。学不来的（灵性、阅历、独特视角）才是精华。
- **坚持写完第一本 30 万字**：新手多半没有掌控长篇的能力，30 万字刚好能把故事讲透且都是精华；写完第一本 30 万，第二本能到 50 万、第三本 100 万。前三到五本多是练手，扑街不灰心，下一本一定更好。

## 二、新手必练的三种能力

### 1. 脑中的画面感
**先有画面，才有文字。** 把脑中演绎的画面 100% 还原成文字，并还原其**情感与氛围**（氛围感/代入感）。
- 调动**五感**：视觉、听觉、嗅觉、触觉、心理。
  > 战场：满地狼藉、硝烟滚滚、烧毁一半的旗帜、血腥味、战友倒地、受伤强撑、两耳嗡鸣。
- 练法：看经典影视，挑最精彩的几幕，用文字还原到自己满意为止。

### 2. 灵感练习
**每天想 10 个开头 / 小故事**，每个 100 字能概括即可。
> 例：女主长期家暴中失手杀夫，自首还是埋尸逃跑？／女主家道中落被仇人养大／女主魂穿成恶毒女配如何自救。
一周攒几十个开头，逐个检视：哪些精彩、哪些乏味、哪些能撑到 10 万甚至 30 万字、能否勾起好奇心。日积月累必出好开头。

### 3. 对故事的建造能力（三张纸法）
- **第一张纸**：脑中零散的、你觉得有趣的情节。
- **第二张纸**：各类人设 + 性格特点。
- **第三张纸**：故事背景、时代背景、几个必须有的场景。
- 然后**打散重组**：让奇怪人设出现在不同场景、搭配不同情节，碰撞新火花。

## 三、大纲（大纲）怎么写

### 大纲 = 一根粗树干
故事是否立得住、是否精彩，看两点：**树干够不够粗壮**、**枝叶够不够繁盛**。

### 大纲要讲清起承转合，展开为：
> 主角的**困境 → 目标 → 阻碍 → 努力过程 → 中途意外 → 意外后的转折 → 最终结局**。
- 本质是**升级打怪 / 屌丝逆袭**：从无到有、从弱到强，波涛起伏才精彩（例：《甄嬛传》女主成长主线）。
- 主线 = 主角人物成长线，所有配角与支线都围绕它展开。
- **主线 + 辅线**：辅线写主要配角的目标/困境/转折/结局；写清人物设定、人物关系、主配角如何互相推动事件（带来机遇还是挑战）。
- **爽点 + 反转**（网文商业属性核心）：研究读者期待点 → 主角受困后小逆袭（爽点）→ 爽点后意想不到的反转 → 反复，高潮迭起。
  > 例：首富装穷被欺辱 → 读者期待"亮明身份打脸"的爽点。
- **名言（写"极度精彩"的觉悟）**：
  - "唯恐天下不乱，乃作家职责所系。"——钱德勒
  - "人物摊上大事后千方百计脱困，即'编排情节'。"——康拉德
  - "我从不写没到穷途末路的人物。"——埃尔金
- **大纲不是死的**：留想象空间，边写边改，让故事更完美。

### 人物小传写法
基础项：身份、年龄、职业、性格、**欲望、目标、惧怕什么**。
- 没有完美的人，**主角的缺点 = 最能出彩的地方**。
- 核心是**反差感**：
  > 极度自恋，但在女主面前极度自卑／白天救死扶伤的好医生，晚上是杀人狂魔／人前三好男人，背地里家暴出轨酗酒赌博。
- 追问：他的目标是什么？为目标会付出什么行动？是否被逼入绝境？过往是好人坏人？做过哪些不为人知的事？这些事如何与配角牵扯？
- **人设占好小说的一大半**，务必重视。观察身边真人的性格/习惯/说话方式/成就与耻辱/极端情境下的心理与外在表现。

## 四、细纲（细纲/章纲）怎么写

**细纲 = 每章或每几章的简单内容概括**，作用是"开枝散叶 + 防止跑偏"（大纲定方向，细纲盯执行）。

### 写法
1. 先定每日更新字数（如 4000 字）。
2. 按经验：4000 字 ≈ **两个小情节**，每个约 2000 字。
3. 用**三五句话**概括后面情节的大致内容，紧扣主题；写的时候在细纲框架内扩充，让章节丰满。

### 关键技巧：切画面 / 转视角制造悬念
> 女配把女主推下山崖 → 写坠崖过程、女主奄奄一息 →（**戛然而止，不交代死活**，吊胃口）
> **切画面到女配**：认定女主死了 → 伪造自杀现场 → 骗家人 → 在女主家耀武扬威、取代地位
> **再切回女主**：没死，如何自救 / 被男主所救。
一份三行细纲，实际能写到 7000 字，可扩充空间极大。

### 习惯
每天更新后花三五分钟捋出**次日细纲**，隔天提笔就写，几乎不卡文。字数越写越多时，细纲能防止"忘记前面写了什么"、防 bug 剧情、防跑偏。

## 五、开头怎么写（黄金三章）

### 核心：不确定就"从狗血开始"
自古套路得人心（《天龙八部》《神雕侠侣》也靠狗血勾人；欧美片开局毁灭世界→小屌丝拯救世界）。有了钩人开头，再往主角身上**加重担**（小到救妻儿父母，大到国家正义）。

### 开篇四要素
**噱头、代入感、氛围感、情绪是否烘托到位。**
- **代入感**：用身边事（例：女主在丈夫手机里发现出轨聊天记录）。真实身边事往往比小说更狗血。
- **不要开篇大段交代背景**：没读者有耐心。用**冲突**或**极具画面感的一段剧情**展开故事；背景在后续用一件事/一段对话简单带过。

### 打磨开头 = 拉满"期待值"
- 女主发现出轨 → 别让她当场大闹离婚（撕逼收尾就没得写了）→ **暗中隐忍、伺机报复**才是读者想看的。
- **像弹簧一样反复拉扯读者的期待神经**，越拉越满，适当时机一击必杀；过程中由主角带出配角与支线（闺蜜、小三、婆婆…），故事被无限拉长。
- **反差感老梗**（开篇即拉期待值）：真假千金、身份互换、互换重生、善良女主穿成恶毒女配、穷养的男主其实是首富（《抓娃娃》）。
- 立意出彩也能靠整体拿捏观众（《饥饿游戏》）；若无强立意，就全靠开头的**后续期待值**。

### 开头必答的问题
- 主角的**动机**是什么？什么阻挡他前进？主角与反派各自的目的？为达目的各自制定了什么计划？
- 你的**噱头和导火索**是什么？读者**为什么**要读下去？
- 希区柯克：好故事就像人生，只是**少了所有无聊的部分**。
- 开篇**每个字都要有意义**，无一句废话——每句话都要问"对后续情节有无推动作用"，没有就是废话。**打磨、打磨、再打磨**，删掉一切无聊无意义的部分。

## 六、story-studio 使用建议

- **构思阶段**（Showrunner / World Architect / Character Designer）：套用"选题材→造场景→天崩开局→活目标→冲突拉扯→暗处关系网"。
- **人设阶段**（Character Designer）：人物小传必含 欲望/目标/惧怕 + 至少一个**反差**或**负资产**。
- **大纲阶段**：先写粗树干（困境→目标→阻碍→努力→意外→转折→结局）+ 主线人物成长线 + 辅线 + 爽点/反转节奏。
- **细纲阶段**（Scene Writer 开写前）：每章三五句话，善用"切画面/转视角"制造悬念（配合 `chapter-hooks`）。
- **开头阶段**（Scene Writer 写第 1 章）：冲突/画面感切入，拉满期待值，杜绝背景倾倒，每句话都要有推动力。
- **长期**：多喂经典作品做参考，追求风格与标签，而非套路堆砌。

> 一句话总结：**给主角天崩开局 + 活的目标 + 强人物拉扯，用画面感落地，靠期待值和反转留人，大纲定方向、细纲防跑偏、开头定生死。**

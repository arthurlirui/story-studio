# 古风世情 — 古代女频短故事创作方法论

> 来源：番茄小说第二十八期作家课堂 · 古风世情类爆款拆解
> 适用题材：宫斗/宅斗、嫡庶矛盾、重生复仇、身份反差、智斗权谋、逆袭打脸
> 定位：与 `webnovel-foundations`（心法）、`chapter-hooks`（钩子）、`climax-design`（爽点）互补，专注古代世情赛道的品类特化技巧

---

## 一、切入点设计：开篇要有抓手

### 三大钩子类型

**1. 悬念钩子 — 用反常信息或未解之谜勾住读者**
- 抛出一句与常识相悖的信息、一个未揭晓的秘密、一个"为什么"。
- 读者被钩子吸引进来后，正文必须**迅速往悬念方向推进**，不能钓鱼不喂饵。
> 例："七皇子百岁宴惊厥，我听见了他的婴语——他说是丽妃把东珠塞进了他嘴里。"

**2. 情绪钩子 — 以高浓度情绪冲突开局**
- 开篇即制造强烈的委屈、愤怒、不甘或紧张感。
- 正文要持续升级冲突，不能开局炸裂然后平淡。
> 例："嫡姐总拿我当绿叶，我的名声坏透了，根本没有正经人家来提亲。"

**3. 身份钩子 — 通过主角的特殊身份或处境引发好奇**
- 重生者、穿越者、特殊能力者、被贬贱奴、被替换的真千金等。
- 身份本身就是"信息差"的来源，天然制造期待。
> 例："上一世我被诬陷私藏厌胜娃娃，满门抄斩。再睁眼，我回到了中秋宫宴那日。"

### 开篇的核心原则
- 导语抛出钩子 → 正文**立刻接住读者的心理预期**，不绕弯、不铺垫背景。
- 开篇不要大段交代世界观/人物前史，用**冲突**或**极具画面感的一段剧情**直接展开。
- 每句话都要问"对后续情节有无推动作用"，没有就是废话。

## 二、人物塑造：行为要有支撑，动机要能自圆其说

### 女主：有成长弧线，不依赖外力
- 女主不能是"等男主来救"的工具人。
- 反击应基于**自身能力**（逻辑推理、对人心的把握、语言交锋），而非机械降神。
- 即使是"以弱胜强"，也要写得有说服力——女主在重重压迫下展现智慧。
- 成长弧线：从隐忍到反击、从被动到主动，每一步都需要合理的触发事件。

### 男主：情感有迹可循
- 不能仅凭"突然就心动了"推动感情线。
- 需要具体的、可信的理由：女主在关键事件中展现的勇气/智慧、两人共患难后自然生发的情感。
- 情感线层层递进、有迹可循 → 读者代入感更强。

### 反派：有独立动机，不能降智
- 反派可以输，但要输得有逻辑；反派可以坏，但要坏得有水平。
- **赋予反派**：合理的动机、清晰的谋略、与主角势均力敌的能力。
- 反派全员降智 → 女主的反击就显得没有分量，爽感大打折扣。
- 读者想看的是主角在真正的强敌面前，凭借智慧赢得漂亮。
- 特别在智斗文、宫斗宅斗文中，反派需要有**应变能力**——当女主反击时，反派能调整策略，形成"矛与盾"的拉锯。

### 人物逻辑自洽检查清单
- [ ] 重要角色的行为是否与其人设相对应、相闭环？
- [ ] 情感转变（恋爱/悔恨/反转）是否有具体的触发事件？
- [ ] 悔恨流中，角色的悔恨是否建立在其性格底色之上？
- [ ] 反派是否有自己的动机，而非纯粹为了害主角而害主角？
- [ ] 人物可以成长、可以转变，但转变的每一步都有逻辑支撑吗？

## 三、剧情节奏与卡点：持续拉满期待

### 核心原则：节奏与看点形成合力
- 每一个重要情节的推进，都应该服务于读者最关心的核心问题。
- 核心问题例如：真相是什么？主角能不能翻盘？坏人会不会受到惩罚？
- 节点前的内容应持续积累读者情绪、提升追读意愿。

### 信息差驱动的节奏模板

**模板 A：信息差打脸型**
```
1. 建立信息差：读者知道真相，但剧中其他角色不知道
2. 蓄力：旁人对女主的羞辱与议论持续积累
3. 读者"智力优越感"：读者一边看一边期待打脸
4. 真相大白：在关键节点，信息差同时消除
5. 反转冲击：读者获得情绪释放
```
> 例：《嫡姐总拿我当绿叶》——读者提前知晓王爷心上人的真实身份，旁人的羞辱在为反转蓄力，同时持续为读者制造"我已经知道了"的优越感。

**模板 B：智斗升级型**
```
1. 陷害出现：女主被诬陷/被置于险境
2. 危机升级：证据看似确凿，女主处境恶化
3. 女主反击：基于逻辑推理和心理把握，逐步辩驳
4. 反派的应变：反派调整策略，冲突升级
5. 最终对决：女主凭借更高层次的智谋获胜
```
> 例：《明棠执棋录》——女主并非依赖外力解围，而是凭借缜密的逻辑推理和对皇帝心理的精准把握逐步辩驳还击。

**模板 C：命运反转型**
```
1. 建立"前世"的惨烈结局（快速闪回/插叙）
2. 回到关键节点：读者知道即将发生什么
3. 女主做出不同选择：读者期待改变命运
4. 原有轨迹被打乱：冲突升级、新危机出现
5. 女主凭借前世记忆+今世成长主导局面
```

### 卡点设置原则
- 每个关键节点前，读者情绪需要**蓄积到位**。
- 读者即将翻页时，抛出**新的悬念**或**即将揭晓的真相**。
- 除了核心爆点，故事应在人物塑造、情节层次或情感深度上具备**额外亮点**（不止一个看点）。

## 四、古代世情赛道的题材谱系

### 高频题材与核心矛盾

| 题材 | 核心矛盾 | 爽点来源 |
|------|---------|---------|
| 宫斗 | 后宫权力争夺 + 皇帝宠信 | 以弱胜强、智斗翻盘、揭露阴谋 |
| 宅斗 | 家族内部嫡庶/妻妾矛盾 | 身份反转、信息差打脸、家族翻盘 |
| 重生复仇 | 前世惨死 → 今生步步为营 | 命运逆转、仇人伏法、身份曝光 |
| 身份错置 | 真假千金/嫡庶互换/替身 | 真相大白、身份回归、反差打脸 |
| 权谋智斗 | 朝堂博弈 + 家族存亡 | 谋略对决、连环计、大局翻盘 |
| 商战/经营 | 古代商业竞争 + 女性自立 | 商业逆袭、经济独立、打脸偏见 |

### 常见设定与钩子
- **特殊能力**：能听懂婴语/动物语言/读心术（信息差来源）
- **重生/预知**：知道前世命运，回到关键节点改变一切
- **身份错位**：双胞胎互换、真假千金、替身代嫁
- **阶级逆袭**：从贱奴/庶女/罪臣之女到高处
- **反差人设**：表面柔弱实则心机深沉 / 表面轻浮实则运筹帷幄

## 五、与 story-studio 创作流程的集成

- **构思阶段**（Showrunner）：套用"选切入点钩子 → 信息差/智斗/命运反转模板 → 设计反派智识水平"
- **人设阶段**（Character Designer）：女主必含"成长弧线起点" + 至少一个"以弱胜强"的能力基础；反派必含"独立动机 + 谋略水平"
- **大纲阶段**：主线 = 女主成长线；辅线 = 反派谋略线 + 情感线；爽点节奏 = 信息差蓄力 → 真相大白 → 新的悬念
- **细纲阶段**（Scene Writer）：每章确认"读者此刻知道什么、不知道什么"，善用信息差切视角
- **开头阶段**：悬念钩子/情绪钩子/身份钩子三选一，开篇冲突直接展开，杜绝背景倾倒

> 一句话总结：**强钩子开局 + 人物逻辑自洽 + 反派有智识 + 信息差蓄力打脸 + 节奏环环相扣。**

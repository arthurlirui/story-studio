# Blurb & Cover Design — 封面 + 简介 双剑合璧

> 来源：番茄小说网官方课《封面简介双剑合璧，作品突围在此一举！》（2025-07-17，阅读 26.6w）
> 用途：story-studio 在**成稿 / 发布 / 包装**阶段，设计高点击率的**封面**和**作品简介**。
> 关系：本技能管"**设计原则 + 简介文案**"；`book-cover-generation` 管"**ComfyUI 出图管线**"。两者配套使用——本技能产出 cover brief 与配色/构图决策，再交给 `book-cover-generation` 生成底图。

> 核心口诀：**"封面是敲门砖，简介是定心丸。"** 封面如磁石（色彩构图瞬间攫取视线），简介如钩线（三行文字定乾坤）。两者相得益彰才能提高点击率。

---

## 一、如何让封面会"说话"

### 1. 核心要素（合格线）
- **清晰**：底图和书名都必须清晰，这是底线。
- **要素突出**：完整体现**作品名、作者名**等重点信息。
- **字号分级**：书名字体一般**大于**作者名。
- **排版避坑**：
  - 书名长度适中 → 适当排版，追求整齐。
  - 书名较长 → 用**不同颜色、不同大小**突出重要文字，避免过度拥挤。
- **书名长度与目标读者匹配**：
  - 脑洞年轻读者 / 快节奏爽文 → 可用**长书名**。
  - 品质类 / 慢节奏内容 → 用**有格调的短书名**。
  - 不绝对，关键看书名风格 × 目标读者 × 内容风格三者契合。
- **封面第一使命**：在一整页众多封面里**一眼锁定读者目光、诱发点击**。先满足这个，再谈创意发挥。

### 2. 设计技巧
- **字体**：用设计感突出作品特色。
- **配色（按品类定调）**：
  - **男频**（历史、玄幻…）→ 偏**深、偏冷**色调。
  - **女频**（现言、古言…）→ 多用**暖色调、偏可爱风**。
  - （附图样例《大汉第一浪》古风言情：暖色/古风人物 CP + 深色氛围底 = 典型女频古言配色。）
- **元素搭配**：在整体美观基础上，适当增加**作品关联元素**（勿堆砌）。
- **提示**：签约后建议用**自定义封面**（比默认模板推荐效果好）；番茄自定义封面工具需作品**字数 ≥ 8w** 才能用。

### 3. 上传规格（番茄平台）
- **尺寸**：建议 **600×800 像素**（竖版）。
- **格式**：jpg / png / jpeg，文件 **≤ 5MB**。
- **入口**：番茄小说网电脑端 或 番茄作家助手 APP → 【创建作品】→【选择封面】→【本地上传】→ 审核通过。

> 给 `book-cover-generation` 的对接要点：底图按 **600×800 竖版**、顶/底留 title-safe space、配色按上面的品类规则设定 palette，书名/作者用**后处理叠字**（不写进 diffusion prompt）。

## 二、简介（作品简介）= 阅读催化剂

### 1. 黄金结构模板（五步）
1. **开篇定调**：首行直接点明故事**核心梗**。
2. **高能开场**：用反套路金句 / 核心冲突 / 灵魂拷问开场。
3. **核心梗概**：**300 字内**讲明故事主线 + 人设亮点。
4. **钩子收尾**：结尾配开放式提问 / 高能预告 / 视觉意象等抓眼元素。
5. **加分技巧**：加一段**小剧场式对话**或亮点剧情。

> 官方优秀示例：女频《岁岁春欢》、男频《长生烬》。

### 2. 避雷指南（三大雷点 + 修改公式）

**① 文艺病** — 堆砌辞藻、无重点
- 删繁就简：**砍掉 50% 形容词**，每句保留 1 个核心动作。
- 具象转化：把"爱恨成空"→"**攥着灭门血证嫁进东宫**"。
- **核心公式：人设标签 + 动词 + 目标**（例：黑莲花嫡女 × 手撕白莲剧本）。

**② 流水账** — 剧透全文、无悬念
- 悬念置换：把"结果"改成"提问"。
- 钩子埋点：结尾留 1 个未解谜团（身世 / 阴谋 / 超自然力量）。
- **核心公式：已知危机 × 未知变数**（例：灭门危机 × 轮回真相）。

**③ 自嗨式** — 忽略读者代入感
- 视角转换：用"**你**"替代"读者"，制造共情。
- 情绪转移：把主题升华转化为**具体爽点**。
- **核心公式：痛点共情 × 解决方案**（例：庶女困境 × 黑莲花逆袭手册）。

> 头号忌讳：**文不对题**。仙侠配言情内容、悬疑配甜宠简介——题材与简介错位，只会把目标读者吓跑。

## 三、封面 × 简介 三大组合公式

1. **「元素呼应」**：封面视觉 ↔ 简介关键词呼应。
   > 现言清新唯美插画风封面 + 简介关键词"校园文学 / 火葬场"。
2. **「情绪同频」**：封面情绪 = 简介情绪。
   > 暗黑系 / 悬疑风封面 + 强冲突简介。
3. **「信息互补」**：封面埋钩，简介填坑。
   > 封面埋钩（疑云 / 红日）+ 简介填坑（灵异 / 游神）。

## 四、story-studio 使用流程

### 触发时机
成稿 / 准备发布 / 用户要求"写简介""设计封面""包装作品"时。

### 分工
- **Showrunner**：定作品定位（品类、目标读者、商业/文学感、是否系列），拍板配色基调与书名长短策略；审核封面 brief 与简介是否"文对题"。
- **Title Designer**：产出书名（配合 `title-design`），并决定书名长短是否匹配目标读者。
- **Scene Writer / Editor**：写简介，套用五步黄金结构，自检三大雷点。
- **Cover Designer（可由 Showrunner 扮演）**：产出 cover brief（品类配色 + 核心视觉 + 竖版构图 + title-safe），交 `book-cover-generation` 出图。

### 简介写作自检清单
- [ ] 首行是否直接点明核心梗？
- [ ] 开场是否高能（金句/冲突/拷问），而非背景铺垫？
- [ ] 梗概是否 ≤ 300 字、讲清主线 + 人设亮点？
- [ ] 结尾是否有钩子（提问/预告/意象/未解谜团）？
- [ ] 是否踩雷：文艺病（形容词过多）/ 流水账（剧透无悬念）/ 自嗨式（无代入感）？
- [ ] 是否"文对题"——简介调性与品类、封面一致？
- [ ] 是否用了"人设标签+动词+目标"或"已知危机×未知变数"或"痛点共情×解决方案"公式？

### 封面设计自检清单
- [ ] 600×800 竖版，jpg/png/jpeg，≤5MB？
- [ ] 书名清晰且字号大于作者名？
- [ ] 配色符合品类（男频深冷 / 女频暖可爱）？
- [ ] 长书名是否用颜色/大小分级避免拥挤？
- [ ] 只有一个核心视觉，未堆砌元素？
- [ ] 与简介构成三公式之一（元素呼应/情绪同频/信息互补）？
- [ ] 书名/作者交给后处理叠字，未写进图像 prompt？

## 五、对《哥伦布计划》的适配

- **品类归属**：西式言情向短篇 → 走**女频暖色/氛围感**路线（狼人/吸血鬼/血族宫廷可叠加**暗黑冷色 + 暖色人物光**做"情绪同频"）。
- **书名**：本系列多为**长网文标题**（如《替嫁狼人首领，他却说等了我三百年》）→ 封面按"长书名"处理：分行 + 颜色/大小分级突出核心梗词（替嫁 / 三百年）。
- **简介公式套用**：
  - 01《替嫁狼人首领…》→ 痛点共情×解决方案：替嫁弃子 × 三百年命定觉醒。
  - 02《我用血养吸血鬼亲王五百年，重生后他跪疯了》→ 已知危机×未知变数：血奴之死 × 重生复仇真相。
- 每本成稿后：Editor 写五步简介 → Cover Designer 出 brief → `book-cover-generation` 出竖版封面 → 三公式校验封面简介是否同频。

> 一句话总结：**封面按品类定色、一眼抓睛、长名分级；简介五步走（定调→高能→300字梗概→钩子→小剧场），避开文艺病/流水账/自嗨式；封面与简介用元素呼应/情绪同频/信息互补锁死同一卖点。**

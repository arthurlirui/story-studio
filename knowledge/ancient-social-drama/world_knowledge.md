# 古风世情 — 世界观与知识库

> 古代女频世情类短故事的通用世界观设定、朝代背景、制度体系、高频场景与题材变体参考。
> 用途：在 story-studio 创作流程中，供 World Architect、Character Designer、Scene Writer 参考和调用。

## 一、朝代与时代背景

### 常用架空朝代设定

古风世情类故事通常采用**架空朝代**，融合多个真实朝代元素：

| 架空设定 | 参考朝代 | 典型特征 |
|---------|---------|---------|
| 盛京/天启 | 明清融合 | 皇权集中、内阁/六部制、后宫等级森严 |
| 长安/洛阳 | 唐风 | 开放繁荣、贵族门阀、诗词文化浓 |
| 建康/金陵 | 魏晋南北朝 | 士族门阀、九品中正、清谈玄学 |
| 临安/汴京 | 宋风 | 商业繁荣、文官政治、市井文化 |

### 常用制度体系

**后宫等级**（适用于宫斗题材）：
```
皇后 → 皇贵妃 → 贵妃 → 妃 → 嫔 → 贵人 → 常在 → 答应 → 官女子
```

**爵位体系**（适用于宅斗/权谋题材）：
```
亲王 → 郡王 → 国公 → 侯 → 伯 → 子 → 男 → 轻车都尉
```

**家族内部结构**（适用于宅斗题材）：
```
老太爷/老夫人（最高辈分）
├── 大房（嫡长）：大老爷 + 大夫人 + 嫡子嫡女 + 庶子庶女
├── 二房：二老爷 + 二夫人 + ...
├── 三房：...
└── 姑奶奶（已出嫁）/ 姨娘（妾室）/ 通房
```

## 三、核心矛盾类型

### 嫡庶矛盾
- 嫡母打压庶女、嫡姐利用庶妹
- 庶子庶女争夺资源、继承权
- 典型结构：庶女隐忍 → 嫡系陷害 → 庶女反击 → 身份反转

### 妻妾矛盾
- 正妻 vs 宠妾
- 继室 vs 原配子女
- 外室/外宅 → 登堂入室

### 真假身份
- 真假千金（被调换）
- 双胞胎互换
- 替身代嫁
- 罪臣之女冒充/被冒充

### 权力博弈
- 后宫争宠 → 皇子之争
- 朝堂党争 → 家族站队
- 外戚 vs 宦官 vs 文官

### 阶级逆袭
- 贱奴/罪臣之女 → 上位
- 庶女 → 王妃/皇后
- 商户女 → 诰命夫人

## 四、常用人物原型

### 女主原型
| 原型 | 特征 | 适用题材 |
|------|------|---------|
| 隐忍谋士型 | 表面柔弱，内心有谋略，步步为营 | 宫斗/宅斗 |
| 重生复仇型 | 前世惨死，今生带着记忆改变命运 | 重生复仇 |
| 特殊能力型 | 有读心/预言/特殊技能，获取信息差 | 宫斗/智斗 |
| 身份错位型 | 真千金/真公主被替换，最终回归 | 真假千金 |
| 底层逆袭型 | 从最低贱的身份一路杀到高位 | 阶级逆袭 |

### 男主原型
| 原型 | 特征 | 适用题材 |
|------|------|---------|
| 冷面王爷型 | 表面冷漠，实则暗中关注/保护 | 宫斗/宅斗 |
| 温润公子型 | 温文尔雅，但关键时刻有担当 | 宅斗/商战 |
| 腹黑权臣型 | 运筹帷幄，与女主势均力敌 | 权谋 |
| 少年将军型 | 武艺高强，直率坦荡，护短 | 边关/战争 |
| 皇帝/太子型 | 身处权力中心，情感与政治纠葛 | 宫斗 |

### 反派原型
| 原型 | 特征 | 关键：不能降智 |
|------|------|--------------|
| 伪善嫡母型 | 表面贤良，暗地算计 | 有手腕、有耐心、善于伪装 |
| 得宠妾室型 | 以色侍人，但心机深沉 | 有情报网、善于利用男主弱点 |
| 嫉妒嫡姐型 | 表面端庄，实则嫉妒女主 | 有自身优势（家世/才学），不甘被超越 |
| 权臣/国丈型 | 以家族利益为名行恶 | 有政治资本、有党羽、有谋略 |
| 白月光/青梅型 | 男主的"白月光"，实则心机 | 善于利用男主的愧疚/旧情 |

## 五、信息差模板库

### 读者知道 vs 剧中人不知道
- 读者知道女主是重生者 → 剧中人以为她是普通庶女
- 读者知道王爷心上人是女主 → 旁人以为王爷看上了嫡姐
- 读者知道七皇子的婴语真相 → 众人以为是惊风

### 女主知道 vs 男主不知道
- 女主知道自己是真千金 → 所有人以为她是养女
- 女主知道前世结局 → 男主不知道她为何突然冷淡

### 女主知道 vs 读者不知道
- 女主暗中布局 → 读者直到真相大白才知道她的计划
- 女主有隐藏身份/能力 → 在关键时刻揭晓

## 六、节奏模板

### 短篇（三万到五万字）节奏
```
第1章：强钩子开局（悬念/情绪/身份三选一）
第2-3章：冲突升级，反派登场，女主陷入困境
第4-5章：女主第一次小反击，但遭遇更大危机
第6-7章：关键转折，女主获得关键信息/盟友
第8-9章：真相大白/身份反转/打脸高潮
第10章：结局，女主站稳脚跟，开放式/HE
```

### 中篇（十万到十五万字）节奏
```
第1-3章：开局 + 第一个冲突环
第4-8章：第一个冲突环解决 + 更大的冲突浮现
第9-12章：伏笔回收 + 身份/情感转折
第13-15章：最终对决 + 结局
```

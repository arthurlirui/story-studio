# 古文虐恋 — 世界观与知识库

> 古代女频虐恋类短故事的通用世界观设定、情感冲突类型、高频场景、人物原型与情绪节奏模板。
> 用途：在 story-studio 创作流程中，供 World Architect、Character Designer、Scene Writer 参考和调用。

---

## 一、朝代与时代背景

### 常用架空朝代设定

古文虐恋类故事与古风世情共用架空朝代体系，但场景侧重**内宅、后院、宫廷内苑**等私密空间：

| 架空设定 | 参考朝代 | 典型特征 |
|---------|---------|---------|
| 盛京/天启 | 明清融合 | 礼教森严、男女大防、婚姻由父母之命 |
| 长安/洛阳 | 唐风 | 风气相对开放，但门第观念极重 |
| 建康/金陵 | 魏晋南北朝 | 士族门阀、联姻政治、年龄差距婚常见 |
| 临安/汴京 | 宋风 | 文官家庭、科举联姻、程朱理学约束 |

### 虐恋常用制度设定

**婚姻制度**：
- 正妻/嫡妻：明媒正娶，三书六礼
- 平妻：特殊情况下可娶（如两房兼祧）
- 妾室/姨娘：纳妾无正式婚礼，地位低于正妻
- 通房/侍妾：最低等，可被随意发卖
- 和离/休妻：古代离婚（和离需双方同意，休书只需男方）
- 师徒姻：师者为尊，弟子需服从

**年龄/辈分设定**：
- 姐弟恋：女主年长男主 6-10 岁，"老妻"羞耻感是核心虐点
- 师徒恋：女主是男主的老师/启蒙者，天然不对等
- 继室/续弦：女主是原配去世后娶的继室，处处被对比

## 二、高频情感冲突场景

### 日常虐感场景（失望积累）

- **独自度过重要时刻**：生产/生病/守岁时，男主不在身边
- **嫁妆/物品被挪用**：女主的嫁妆被转送给女配
- **被比较**：女主的行为/才华/容貌被拿来与女配对比
- **被忽视**：女主的请求/需要被当作"无关紧要"
- **被替代**：女主的位置/物品/角色被女配轻易取代
- **被羞辱**：公开场合被贬低，男主不维护
- **独自面对危机**：危难时刻，男主选择了保护女配

### 关键转折场景

- **撞见真相**：女主意外撞见男主对女配的温柔，或听到男主对女主的真实评价
- **生死关头**：女主面临生死危机，男主的反应暴露了真实态度
- **身份揭露**：女主的真实身份/付出被揭露，但已经太晚
- **决定离开**：女主在某个普通时刻，平静地做出了离开的决定
- **最后一面**：女主离开前，男主仍然毫无察觉

### 追妻场景（爽感释放）

- **发现她走了**：男主发现女主已经离开，开始慌乱
- **追悔莫及**：男主意识到自己失去了什么，疯狂寻找
- **被拒绝**：男主追到女主面前，被冷静拒绝
- **角色反转**：曾经女主做的事，现在男主在做（熬药/等待/讨好）
- **当众表白被拒**：男主在公开场合表白，女主不为所动

## 三、核心虐感类型

### 不对等付出
- 女主付出 100% → 男主回报 10%
- 具体表现：陪伴、关心、资源、时间、情感的不对等
- 关键写法：不是"她对他好，他对她不好"，而是"她为他做的每一件事，他都为别人做了，唯独没为她做过"

### 身份羞耻
- 姐弟恋："老妻"羞耻（男主娶她时的羞耻感）
- 师徒恋：师者身份被当作"理应付出"
- 替身恋：被当作"替代品"，真实情感不被看见
- 庶女/婢女身份：因出身被轻视，付出被当作理所当然

### 被替代
- 女配怀孕 → 他全程守护，女主怀孕 → 他毫不在意
- 女配的孩子 → 他亲手教养，女主的孩子 → 无人问津
- 女配的喜好 → 他记在心上，女主的喜好 → 他从未问过
- 女配的嫁妆 → 他精心准备，女主的嫁妆 → 被他转送别人

### 极致忽视
- 女主悄然策划离开，男主始终毫无察觉
- 女主的情绪变化、身体状况、精神状态，男主从未注意
- 这种"忽视为最高级的虐"——不是恨，是不在乎

## 四、情感错位对比模板

### 对比闭环写作法

**核心**：同样的事件，前后形成对比，让读者感受到"曾经的她"和"现在的她"的差异。

```
事件类型         前期（女主主动）              后期（男主主动）
─────────────────────────────────────────────────────
送礼物/关心      她亲手熬药，他不屑一顾         他亲自下厨，她不需要
等待             她等他到深夜，他去了别人那里    他在她门前等一夜，她早已安睡
求助             她求他帮忙，他敷衍了事          他主动帮她，她已有了别人
表达爱意         她写的情诗，他从不看            他写的信，她原封不动退回
公开场合         她当众维护他，他默不作声        他当众表白，她淡然拒绝
```

### 情感错位的节奏控制

```
前期（60%）：女主的付出 vs 男主的忽视
  → 每一个"她为他做的事"都是未来的"对比素材"

转折点（10%）：女主决定离开
  → 不是某个大事件，而是"积累够了"

后期（30%）：男主的追妻 vs 女主的平静
  → 每一个"他做的事"都在呼应前期"她做过的事"
```

## 五、常用人物原型

### 女主原型
| 原型 | 特征 | 关键虐点 |
|------|------|---------|
| 年长付出型 | 比男主年长，以师者/姐姐身份付出 | 年龄羞耻、付出不被看见 |
| 替身隐忍型 | 被当作某人的替身，付出真心 | 被替代、被比较 |
| 庶女/婢女型 | 出身低微，因身份被轻视 | 阶级不对等、付出被当作理所当然 |
| 重生看淡型 | 前世被辜负至死，今生不再投入 | 前世虐感与今生冷感的对比 |
| 身份错位型 | 双胞胎/真假千金，命运被交换 | 命运的残酷对比 |

### 男主原型
| 原型 | 特征 | 需要埋设的"被爱证据" |
|------|------|---------------------|
| 冷漠王爷型 | 表面冷硬，对女主漠不关心 | 曾在她最无助时伸出援手 / 某个瞬间展现了真诚 |
| 白月光型 | 心里有白月光，女主是替代品 | 对白月光的深情本身是魅力（但后来才发现女主的真心） |
| 利益至上型 | 娶女主是政治联姻，毫无感情 | 在某个关键时刻曾展现过原则/底线 |
| 年少轻狂型 | 比女主年轻，不成熟，羞于"娶了老妻" | 曾经的少年气/单纯/才华真实存在过 |
| 愧疚型 | 因某种原因亏欠女主，但不知如何弥补 | 曾经有过温柔的时刻，但被愧疚/骄傲掩盖 |

### 女配原型
| 原型 | 特征 | 不能写成"纯粹的恶" |
|------|------|-------------------|
| 白月光/青梅 | 男主心中的白月光 | 她本身可能并不坏，但男主对她的偏爱本身就是虐点 |
| 娇弱表妹 | 以柔弱博取同情和保护 | 她的"弱"是武器，但并非完全伪装 |
| 得宠妾室 | 以色侍人，善于争宠 | 她的手段有逻辑，不是纯粹的无理取闹 |
| 家族棋子 | 被家族安排嫁给男主的贵女 | 她也是受害者，与女主形成对比 |

## 六、情绪曲线模板

### 短篇（三万到五万字）虐恋情绪曲线

```
第1章：破碎细节切入（甜蜜回忆→现实虐真相）
第2-3章：日常失望积累，情感对比场景密集铺排
第4-5章：关键转折事件，女主决定离开
第6-7章：女主悄然布局，男主仍无察觉（极致忽视的高潮）
第8-9章：女主离开，男主发现真相，开始追妻
第10章：结局——女主已从容，男主追悔莫及（OE/HE可选）
```

### 中篇（十万到十五万字）虐恋情绪曲线

```
第1-3章：虐感底色的建立 + 情感对比场景
第4-6章：第一个小高潮（关键事件触发情感动摇）
第7-9章：日常失望继续积累 + 女主开始悄然谋划
第10-12章：转折点（女主决定离开）+ 离开前的"最后一面"
第13-15章：追妻线 + 女主的从容 + 结局
```

### 情绪密度控制
- **虐感章节**：每章至少 2-3 个情感对比/不公平感场景
- **过渡章节**：穿插松弛治愈的日常，但不能让读者觉得"女主原谅了"
- **爽感章节**：每章至少 1-2 个"他有被虐到"的瞬间

## 七、参考作品库

### 番茄爆款范例
- 《寒灯照旧梦，她辞高堂月》— 姐弟恋 + 日常回忆碎片 + 对比描写 + 悄然离开
- 《被强夺后位后，我不争了，他却急了》— 重生 + 身份错置 + 30%虐感铺垫 + 70%追妻爽感
- 《窃我娘亲尊荣，我赐他们满门覆灭》— 阶级逆袭 + 复仇 + 明暗双线

### 经典参考
- 《东宫》— 虐恋 + 身份反转 + 悲剧结局
- 《步步惊心》— 穿越 + 虐恋 + 命运无力感
- 《香蜜沉沉烬如霜》— 虐恋 + 误会 + 追妻
- 《知否》的某些支线 — 隐忍 + 不对等婚姻 + 女性觉醒

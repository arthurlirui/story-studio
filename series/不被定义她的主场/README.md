# 《不被定义她的主场》系列工程

## 系列定位

《不被定义她的主场》是一个面向女频长篇网文批量创作的女本位系列宇宙/知识库工程。

核心口号：

> 女子就是好！全体目光朝女主看齐！  
> 她可以驰骋职场、逆袭人生、坚守热爱、打破困境，也可以遵从内心、活出自在。  
> 她的人生、性格、命运、身份、年龄、能力和金手指，都不该被他人一句话定义。

## 系列核心

所有故事必须以女性视角为核心，以女性成长、女性力量、女性自主为叙事主线。

本系列不是“女主谈恋爱顺便成长”，而是：

```text
女主拥有自己的目标
  → 世界用某种标签定义她
  → 她被压制、误解、规训、消耗
  → 她用行动重写规则
  → 所有人重新看见她
  → 她拿回人生主场
```

## 题材范围

题材不限，可包括：

- 现言职场、都市成长、离婚后事业线、女性创业。
- 古言宅斗、女官、商户女、通房/庶女/寡妇身份反转。
- 年代文、六零七零、工业女性、女工匠、家国建设。
- 仙侠、幻言、女修、妖族、神女、逆天改命。
- 无限流、废土、末世、悬疑、轻科幻、虚实穿梭。
- 年龄反差：小女孩、少女、成熟女性、老太太重新上场。

## 必须满足的三项条件

### 1. 女本位内核

女主为绝对核心。她人格独立，目标清晰，剧情围绕她的成长、奋斗、逆袭、选择展开。

### 2. 拒绝刻板标签

女性角色可以飒爽、温润、野心勃勃、特立独行、敏感、迟钝、锋利、不完美、有棱角。摒弃傻白甜、恋爱脑、工具人、单一贤妻良母等扁平形象。

### 3. 明确“不被定义”的立意

每本书必须选择至少一个“不被定义”的方向：

- 不被时代定义。
- 不被身份定义。
- 不被年龄定义。
- 不被职业定义。
- 不被婚恋定义。
- 不被出身定义。
- 不被天赋/金手指定义。
- 不被失败定义。
- 不被身体/疾病/缺陷定义。

## 与《破镜之后》的差异

《破镜之后》核心是旧情、旧伤、迟来追悔与原谅权。  
《不被定义她的主场》核心是女主突破标签、重写规则、建立自我主场。

爱情可以存在，但不是女主价值的判定器。

# 章节生成模板

## 输入

- 书名：
- 卷名：
- 章节号：
- 章节标题：
- POV：女主 / 女性盟友 / 对手 / 群像
- 本章女主目标：
- 本章被施加的定义/标签：
- 本章行动：
- 本章爽点：
- 本章小高潮：
- 本章结尾钩子：

## 写作要求

1. 开头 300 字内出现具体处境或冲突。
2. 本章必须让女主做出一个选择或行动。
3. 爽点必须来自女主能力、判断、韧性或新规则。
4. 避免空喊女性独立，要写具体细节和结果。
5. 若有男性角色，不得抢走女主主线。
6. 每章结尾要有推进：信息、关系、权力、能力、主场之一发生变化。

## 章节结构

### 1. 开场困境

世界如何定义她？

### 2. 女主行动

她不接受定义，做了什么？

### 3. 冲突升级

旧规则如何反扑？

### 4. 爽点兑现

她用什么能力/策略/关系破局？

### 5. 小高潮/钩子

本章最后留下什么新变化？

## 输出格式

```markdown
# 第X章 章节名

正文……
```

## 质量检查

- [ ] 女主是否有主动性？
- [ ] 标签是否具体？
- [ ] 爽点是否清晰？
- [ ] 女性角色是否不工具化？
- [ ] 结尾是否有小高潮？

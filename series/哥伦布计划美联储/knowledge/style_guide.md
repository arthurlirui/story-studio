# 写作风格指南：金融帮派叙事专项

> 本指南为《哥伦布计划美联储》系列提供金融帮派叙事的写作风格指导。

---

## 一、系列统一叙事风格

### 1.1 叙事基调

**金融史诗感，但不故作高深。**

- 金融概念要真实，但要通过人物和情节自然展现，不是教科书。
- 家族权力斗争要有"教父"式的庄严感，但人物要有温度。
- 让金融的"冷酷"与家族的"忠诚"形成张力。

**三个关键词**：精准、冷冽、暗涌。

- **精准**：金融术语必须准确，不能糊弄。
- **冷冽**：叙事调性保持冷静，不煽情，让权力博弈本身说话。
- **暗涌**：表面平静的会议室对话，暗藏杀机。

### 1.2 叙事距离

- 第三人称有限视角，跟女主走。
- 不做全知全能的上帝叙事。
- 金融信息通过人物的判断和决策自然呈现。

### 1.3 时间感

- 金融世界的时间节奏：暗流会议的倒计时、清算日的逼近、季度报表的截止日期。
- 用金融的时间节奏构建叙事节奏。

## 二、金融概念的叙事法则

### 2.1 "金融即帮派"的翻译法则

凡是金融术语出现，必须同时给出"帮派版"的理解。

| 金融术语 | 帮派翻译 | 叙事示例 |
|---------|---------|---------|
| 利率上调 | 贡金比例提高 | "二十七人会议决定提高贡金比例。这意味着，下个季度，所有依赖借贷的家族，都得交出更多的血。" |
| 银行挤兑 | 家族信用被攻击 | "今天早上，六个城市的存款人同时涌入了我们的银行。这不是巧合。这是有人在背后告诉他们——我们没钱了。" |
| 做空 | 金融暗杀 | "有人在市场上大量借入我们的股票，然后抛售。他们不是想赚钱。他们是想让我们死。" |
| 并购 | 家族吞并 | "他们要的不是合作。他们要的是控制——用我们的钱，买我们的命。" |

### 2.2 金融场景的"质感法则"

每个金融场景至少覆盖 3 种感官，至少 1 个金融特写：

```
会议室：长桌的核桃木纹路，雪茄的烟雾在天花板下盘旋，27个座位只坐了21个人——有6个空位，那是背叛者的位子。

清算室：壁炉里的火映在墙上，会计把账本摊开。每翻一页，就有一个人皱眉。翻到最后一页，有人站起来，说："我们欠了多少钱？" 会计没有抬头："不是钱。是协定。"

暗流会议密室：地下室，没有窗户，只有一盏灯。六个人围坐在圆桌旁。桌上放着一把老式天平——一端是黄金，另一端是血。这是Davison家族的规矩：每次会议开始前，必须称量。
```

## 三、家族权力的叙事风格

### 3.1 权力展示的法则

权力不是"说"出来的，是"做"出来的。

**权力展示的三种方式**：
1. **沉默展示**：族长不说话，但所有人都在等他先说。
2. **细节展示**：他走进房间，所有人的椅子都往后挪了一寸。
3. **决策展示**：他用一句话，改变了一个家族一个季度的命运。

**禁止**：旁白说"他是最有权势的人"——要用场景证明。

### 3.2 家族对话的风格

- 家族内部对话：正式、克制、暗含等级。
- 家族间谈判：针锋相对，但不失体面。每句话都有两层意思。
- 家族对下属：简洁、命令式、不解释。

**好的家族对话示例**：

> "父亲，我们被做空了。"
>
> 老人没有抬头，继续削他的苹果。刀锋贴着果皮，旋转，一圈，两圈，三圈。
>
> "谁？"
>
> "不确定。可能是 Morgan，也可能是 Warburg。"
>
> 苹果皮断了。老人把刀放在桌上。"那就做空他们所有人。"
>
> "但我们没有那么多——"
>
> "你有。你只是不知道你有。"

## 四、金融爽点的叙事法则

金融爽点不同于武力爽点。它不是"一拳打翻对手"，而是"让对手的资金链在72小时内断裂"。

**金融爽点的四步法**：
1. **蓄力**：女主发现有人在暗中攻击她的家族。她开始调查。
2. **布局**：她不动声色地布置反击——秘密收购、媒体操控、离间对方盟友。
3. **引爆**：在暗流会议或清算日，她当众揭穿对方的操作，同时发动金融反击。
4. **余波**：对方家族陷入危机，但女主没有"赶尽杀绝"——她提出了一个条件。

**禁止**：金融爽点写成"女主突然发现了一个漏洞，然后赢了"——必须有布局过程。

## 五、感情线的叙事风格

在金融帮派世界中，感情的表达方式是克制的、含蓄的，但同时也是致命的。

**"金融即感情"法则**：
- 他给了她一笔秘密贷款，不是为了帮助她，而是为了让她欠他一个人情。
- 她在暗流会议上投了他一票，不是因为支持他的提案，而是因为她知道如果他输了，他会死。
- 他送她的不是花，而是一份"暗流纪要"的副本——这意味着他背叛了自己的家族。

**好的感情场景示例**：

> 他把文件推到她面前。"这是下季度暗流会议的投票意向。我的家族，加上你的家族，就够票数了。"
>
> 她没有碰文件。"你为什么要给我这个？"
>
> "因为如果我不给，你明天就会输。"
>
> "你知道这意味着什么吗？"
>
> "我知道。"他站起来，走到门口，停下来。"我背叛了我的家族。但我不后悔。"
>
> "为什么？"
>
> 他没有回头。"因为你的家族，值得我背叛一次。"

## 六、每个故事必含的元素

| 元素 | 说明 | 必须出现 |
|------|------|---------|
| 密室会议 | 至少一次家族密室或二十七人会议的封闭场景 | ✅ |
| 金融操作 | 至少一次完整的金融操作展示（挤兑/做空/利率操控/收购） | ✅ |
| 家族血誓 | 至少一次家族血誓的仪式、提及或违反 | ✅ |
| 清算日 | 至少一次清算日的场景或提及 | ✅ |
| 历史回声 | 至少一个真实历史事件的"帮派映射"版本 | ✅ |
| 女性的力量 | 女性角色必须展示真实的金融/政治权力 | ✅ |

---

## 七、禁止的叙事陷阱

1. ❌ 金融概念写成"看不懂的魔法"——每个金融操作必须让读者"理解"。
2. ❌ 家族写成"纯粹的邪恶"——每个家族都有内在的荣誉感和忠诚。
3. ❌ 女性角色写成"花瓶"或"恋爱脑"——女性必须有独立的金融权力线。
4. ❌ 暗杀写得太频繁——暗杀是"最后手段"，不是"日常手段"。
5. ❌ 历史写成"阴谋论"——所有金融体系设定必须基于真实历史逻辑。
6. ❌ 感情线写成"霸总爱上我"——感情必须在金融博弈中自然产生。

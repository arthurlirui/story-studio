# 冲突引擎：金融战争模板

> 本文件提供《哥伦布计划美联储》系列可复用的冲突模板。
> 核心原则：金融手段是主要武器，暗杀是辅助手段。智斗优先于武斗。

---

## 一、金融战争的总原则

```
金融战争 ≠ 开枪杀人
金融战争 = 让对手的资金链断裂，让对手的信用崩溃，让对手的盟友背叛
```

**关键法则**：
- 一场金融战争的核心是"让对手的资产变得一文不值"。
- 最优雅的金融谋杀：不是杀一个人，而是杀一个家族的信誉。
- 暗杀是"最后的手段"，不是"首选手段"——因为暗杀一个家族族长，只会让他的继承人复仇。

---

## 二、金融战争武器库

### 1. 挤兑（Run on the Bank）
- **原理**：散布谣言，让一个家族银行的存款人同时提款，导致银行破产。
- **帮派场景**：一个家族在暗中散布另一个家族"即将破产"的谣言→存款人恐慌→挤兑→家族信用崩溃。
- **防守手段**：证明自己资金充足，或从其他家族获得紧急贷款。
- **故事张力**：女主必须在72小时内筹集足够的资金来阻止挤兑。

### 2. 做空（Short Selling）
- **原理**：借入对手的资产卖出，然后散布负面消息让资产价格下跌，低价买回归还，赚取差价。
- **帮派场景**：一个家族通过金融衍生品（CDS/CDO）做空另一个家族的核心资产。
- **防守手段**：反向操作——拉升资产价格，让做空者爆仓。
- **故事张力**：女主发现有人在大量做空她家族的基金，她必须在对手平仓前找到资金反击。

### 3. 利率操控（Interest Rate Manipulation）
- **原理**：通过影响二十七人会议，提高或降低利率，改变全美地下世界的资金成本。
- **帮派场景**：一个家族推动升息，让依赖借贷的竞争对手家族资金链断裂。
- **防守手段**：在二十七人会议中争取足够票数，阻止升息/降息。
- **故事张力**：女主是二十七人会议中唯一一个反对升息的家族代表，她必须说服其他家族的代表。

### 4. 收购/并购（Hostile Takeover）
- **原理**：通过二级市场大量收购对手家族的股份，达到控制权。
- **帮派场景**：一个家族秘密收购另一个家族的核心企业股份，试图夺取控制权。
- **防守手段**：毒丸计划、白衣骑士（找到另一个家族作为友好收购方）。
- **故事张力**：女主发现她家族的股份正在被一个匿名买家大量收购，她必须找到这个买家是谁。

### 5. 信息战争（Information Warfare）
- **原理**：获取/泄露对手的秘密信息，破坏其信用或制造内部矛盾。
- **帮派场景**：一个家族获取了"暗流纪要"的副本，泄露给媒体。
- **防守手段**：找出泄密者，控制信息扩散。
- **故事张力**：女主发现她家族的"暗流纪要"副本失窃，她必须在信息被公开前找回。

### 6. 联盟背叛（Alliance Betrayal）
- **原理**：说服对手的盟友背叛，使对手孤立。
- **帮派场景**：一个家族向另一个家族提供更好的条件，使其背叛当前的联盟。
- **防守手段**：提前发现背叛迹象，提供更好的条件挽回。
- **故事张力**：女主在联盟会议上发现，她最信任的盟友正在与敌人秘密谈判。

## 三、短篇金融战争模板（8-12章）

### 模板 A：挤兑 + 反击

```
第1章：金融危机开场 → 女主家族的银行出现异常提款
第2章：阴谋浮现 → 她发现挤兑是另一个家族在背后操纵
第3章：男主登场 → 他是敌对家族的人，但他主动提供了关键信息
第4章：第一次反击 → 女主用自有资金暂时稳住局面
第5章：危机升级 → 敌人加大攻势，更多存款人涌入
第6章：信任危机 → 女主发现家族内部有叛徒
第7章：男主的选择 → 他必须在家族和女主之间做出选择
第8章：金融反击 → 女主用一场精心策划的反做空，让敌人自食其果
第9章：叛徒揭露 → 家族内部的叛徒被揭穿
第10章：结局 → 家族重组，女主获得更高的地位，感情线收束
```

### 模板 B：利率战争 + 联盟博弈

```
第1章：暗流会议 → 二十七人会议即将投票决定利率，女主家族面临巨大压力
第2章：站队 → 六大家族分成两派，女主必须在会前争取足够票数
第3章：秘密交易 → 女主与一个家族进行秘密谈判，但被对方设了陷阱
第4章：信息泄露 → 女主的谈判内容被泄露，她的家族信用受损
第5章：男主介入 → 他是敌对家族的儿子，但他发现了自己家族的不正当手段
第6章：反转 → 女主用泄露的信息反向追踪，找到了真正的幕后黑手
第7章：投票 → 二十七人会议投票，女主在最后一刻逆转了局势
第8章：结局 → 利率决议通过，金融格局重新洗牌，感情线收束
```

## 四、长篇金融战争模板（四卷）

### 第一卷：火焰初燃
- 女主家族遭受第一次金融攻击。
- 她发现攻击的背后是另一个家族在操纵。
- 她开始学习金融战争的规则。
- 与男主第一次因金融立场产生冲突。

### 第二卷：暗流涌动
- 攻击升级，女主的家族面临严重危机。
- 她必须与潜在的盟友谈判，但所有人都在观望。
- 她的家族内部出现裂痕——有人主张投降，有人主张反击。
- 感情线深化，但金融立场不同导致冲突。

### 第三卷：血色清算
- 女主发动全面反击，使用了所有金融武器。
- 战争波及所有六大家族——没有人能置身事外。
- 二十七人会议被迫介入，但会议本身已经分裂。
- 男主在关键时刻做出了选择——背叛家族支持女主。

### 第四卷：新秩序
- 战争结束，金融格局重新洗牌。
- 女主必须在旧秩序的废墟上建立新的规则。
- 但不是"她赢了，别人输了"——她必须找到一种让所有家族共存的方式。
- 感情线收束，但不是在"王子与公主幸福生活"的意义上——而是在"两个人在权力与爱的平衡中，找到了自己的位置"。

## 五、禁止的冲突模式

1. ❌ 纯武力解决——不能用"枪杀所有对手"解决金融问题。
2. ❌ 金融概念当噱头——不能只是"提到"做空/挤兑/利率，而不解释运作逻辑。
3. ❌ 女主靠男主拯救——女主的金融反击必须基于她自己的智慧和判断。
4. ❌ 所有对手都是蠢货——金融战争的对手必须有同等水平的金融智慧。
5. ❌ 金融战争被简化为"数字游戏"——必须有真实的人性代价（破产、自杀、家族分裂）。
6. ❌ 黑白分明——每个家族在金融战争中都有自己的"合理性"。

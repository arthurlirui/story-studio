## 世界观核心

- **穿越机制**：5种方式（濒死/醉酒/梦境/被袭/器物触发）
- **时代锚点**：7个历史时期（秦汉/三国/隋唐/两宋/元明/明清/明郑）
- **知识金手指**：武器军事、商业金融、农业民生、策略治理四大体系
- **核心矛盾**：现代思维 vs 古代规则、知识优势 vs 物质限制

# 《重生之动物变身》系列总纲

> 6部网文小说，每部12章 × ~5000字/章 = ~6万字/部 | 总计 ~36万字
> 核心卖点：动物萌宠 + 美女暧昧 + 社死搞笑 + 媒体狂欢

---

## 一、系列概念

### 1.1 核心设定

社会底层失意者（学生/职员/无业者）因意外（药物/雷劈/溺水等），灵魂转移到萌宠身上。凭借人类灵魂+动物身体的反差，意外接近了做人时永远无法接触的美女（校花、明星、网红、女总裁等），做出各种"社死"但爆红的行为，引发媒体追逐和更多美女的"亲密接触"。

### 1.2 世界观规则

1. **变身机制**：灵魂转移，动物身体保留人类意识
2. **变身条件**：濒死体验 + 强烈执念（对美女的渴望/对生活的不满）
3. **动物限制**：无法说话（但能理解人类语言），保留动物本能（如摇尾巴、舔人）
4. **人类认知**：动物身体做出"人类行为"时，会被认为是"天才动物"
5. **媒体效应**：每个主角都会因为奇特行为成为网红动物
6. **暧昧边界**：动物身份提供了"安全距离"——可以做人类不敢做的事

### 1.3 统一叙事节奏

```
第1-3章：变身觉醒 → 适应动物身体 → 首次接触美女
第4-6章：社死名场面 → 意外走红 → 更多美女出现
第7-9章：媒体狂欢 → 身份危机 → 情感升温
第10-12章：真相边缘 → 终极选择 → 结局反转
```

## 三、写作风格规范

### 3.1 网文风格要点

- **标题党美学**：每章标题要夸张、有冲击力、让人想点
- **爽点密度**：每章至少3个爽点（被美女关注/社死爆红/打脸逆袭）
- **笑点密度**：每章至少3个笑点（动物本能vs人类灵魂的反差）
- **暧昧描写**：用擦边但不越界的语言描写亲密接触
- **第一人称**：以主角动物视角叙述，增强代入感
- **弹幕体**：适当加入"弹幕式"内心吐槽

### 3.2 每章结构模板

```
[章节标题]（夸张网文风）
[插图提示]（AI绘图prompt）
---
[开场钩子] 200字 — 抓人眼球的开场
[场景展开] 1500字 — 本章核心场景
[爽点×3] 穿插分布 — 被美女关注/社死爆红/打脸
[笑点×3] 穿插分布 — 动物本能搞笑/内心吐槽
[暧昧场景] 800字 — 与美女的亲密接触（擦边）
[章末悬念] 200字 — 让人想看下一章
---
总计：~5000字/章
```

### 3.3 插图规范

每章配一张插图，AI绘图prompt格式：
```
[动漫风格/精致插画]，[场景描述]，[动物特征]，[美女特征]，[氛围]，高画质，细节丰富
```

### 3.4 暧昧描写尺度

- ✅ 允许：肢体接触、近距离观察、意外触碰、动物本能行为被误解
- ✅ 允许：内心独白表达渴望、用动物视角"欣赏"美女
- ❌ 禁止：露骨的性描写、直接性行为、未成年角色
- 原则：擦边但不越界，暧昧但不低俗

---

## 四、角色命名规范

- 主角：真实姓名 + 变身前身份 + 性格标签
- 女主：真实姓名 + 身份 + 性格反差
- 配角：功能性命名（如"经纪人小王"、"粉丝A"）
- 动物名字：女主给动物起的名字（与主角本名形成反差）

2. `novels/0X_动物名/final.md` — 完整版
3. `novels/0X_动物名/characters.md` — 角色档案

# 《重生之动物变身》写作风格指南

## 一、核心风格

### 1.1 网文体
- **标题党**：每章标题要夸张、有冲击力、让人想点
- **第一人称**：以主角动物视角叙述，增强代入感
- **弹幕体内心独白**：用括号/破折号插入内心吐槽
- **节奏快**：不拖沓，每章有推进

### 1.2 爽点公式
```
被美女关注 + 社死爆红 + 打脸逆袭 = 爽点三连
每章至少3个爽点，均匀分布
```

### 1.3 笑点公式
```
动物本能 vs 人类灵魂 = 反差萌笑点
内心狂喜 vs 动物面瘫 = 表情包笑点
人类行为 vs 动物身体 = 笨拙笑点
每章至少3个笑点
```

### 1.4 暧昧公式
```
动物身份提供"安全距离" → 可以做人类不敢做的事
美女对动物不设防 → 亲密接触自然发生
擦边但不越界 → 暧昧但不低俗
每章至少1个暧昧场景（800字左右）
```

## 三、语言规范

### 3.1 叙述语言
- 口语化，像在跟朋友讲故事
- 大量使用短句，节奏明快
- 适当使用网络流行语（但不滥用）

### 3.2 内心独白
- 用破折号或括号标注
- 吐槽要犀利、自嘲要到位
- 示例："——我发誓，我真的不是故意的！（但确实很爽）"

### 3.3 暧昧描写词汇库
- 视觉：白皙、修长、曲线、锁骨、腰肢
- 触觉：柔软、温热、光滑、细腻
- 嗅觉：香气、体香、奶香、花香
- 动作：蹭、贴、靠、依偎、蜷缩
- 原则：用暗示不用明写，用感受不用动作

### 3.4 禁止事项
- ❌ 露骨的性描写
- ❌ 直接性行为
- ❌ 未成年角色（女主必须18+）
- ❌ 低俗下流的词汇
- ❌ 物化女性的粗暴描写

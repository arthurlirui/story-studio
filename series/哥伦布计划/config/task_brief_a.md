# 故事设计任务书 — 团队 A

## 任务
为"哥伦布计划"系列设计 3 个故事的详细方案：
- **故事 01《月下替身》**：狼人 + 替嫁闪婚 → 先婚后爱
- **故事 02《血吻追赎》**：吸血鬼 + 追妻火葬场
- **故事 03《暗夜逃亡》**：Mafia + 带球跑

## 对每个故事，需要输出以下内容：

### 1. 故事概要（200字）
一句话 + 核心冲突 + 结局走向

### 2. 核心钩子分析
- **开篇钩子**（前三章如何抓住读者）
- **情绪钩子**（虐点/甜点/爽点的布局）
- **悬念钩子**（读者的"然后呢？"在哪里）

### 3. 明线与暗线
- **明线**：读者看到的故事（表层情节推进）
- **暗线**：推动情节的隐藏力量（身世秘密/权力斗争/复仇计划/前世因果）

### 4. 章节目录 + 每章爽点（8-12章）
每章标注：
- 章节标题（吸引人）
- 核心事件
- 🎯 爽点/爆点是什么
- ⚡ 情绪标签（甜/虐/爽/燃/悬疑）

### 5. 前三章详细钩子设计（最关键！）
- **第1章**：开篇必须在前 500 字内建立"不可不读"的钩子
- **第2章**：强化矛盾，建立关系张力
- **第3章**：第一次重大转折/反转，读者彻底入坑

### 6. 人物弧光
- 女主：从什么状态 → 成长为什么状态（强调"觉醒"与"反击力度"）
- 男主：核心魅力 + 核心缺陷 + 转变

## 创作红线
- 西方设定为基础（狼人/吸血鬼/Mafia家族）
- 女主必须"觉醒+反击"，拒绝被动憋屈
- 避免中式设定、东方玄幻、宗教话题
- 前三章必须有强力钩子

## 输出格式
将 3 个故事的设计写入 `/data/openclaw/workspace/story-studio/series/哥伦布计划/ideas/stories_01_03.md`
使用清晰的 Markdown 标题层级。

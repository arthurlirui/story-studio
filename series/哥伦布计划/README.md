## 二、作品规范

### 篇幅
- 短篇：8-12 章，每章 2000-3500 字
- 总字数：2-4 万字

### 结构铁律
- **前 3 章**：必须完成"钩子三连击"——开篇危机 → 身份/关系确立 → 第一次转折
- **每章末尾**：必须有悬念/反转/情绪高点
- **中段**：至少 1 次"爽点爆发"（打脸/身份揭露/反击成功）
- **结尾**：情感收束 + 人物弧光完成

### 人物要求
- 女主必须"有脑子"：拒绝傻白甜，要有主动性和成长
- 男主必须有"苏感"：强大但有缺陷，冰冷但有裂痕
- 配角不为工具人：每个配角有自己的动机

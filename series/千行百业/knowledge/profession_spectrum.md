# 职业谱系库：千行百业全方向覆盖

> 每个职业天然自带一个"人性冲突"和一种"叙事质感"。
> 本库按赛道两大方向排列，供创作时快速查选。
> 使用规则：所有职业必须为现代真实职业，禁止官场/仕途、玄幻修仙、穿越重生。感情线与职业线必须交织，职业细节须经调研而非编造。

---

## 方向一：特色行业风貌

> 职业本身是最大的钩子——读者因为好奇这个行业而翻开。
> 核心要求：职业细节即爽点，每本书读完读者获得一个"内部视角"的行业认知。
> 参考作品气质基准：《机长先生，雷雨请绕飞》《起底观心术》《烈酒玫瑰》

---

### A1. 文物修复师

- **人性冲突**：守护千年 vs 被时代遗忘——她修好了文物，却修不好自己被历史忽略的人生
- **叙事质感**：在博物馆深处，与千年对话，指尖触摸的是时间本身
- **核心场景**：
  1. 修复室里屏息凝神地清洗一幅残损绢画，窗外四季更替她浑然不觉
  2. 文物库房盘点，独自面对上千件未修复的文物，感到这辈子也修不完
  3. 考古现场紧急提取刚出土的脆弱文物，和时间赛跑防止氧化
  4. 修复成果展开幕，观众赞叹文物之美，无人提及修复者的名字
  5. 与老修复师的师徒传承——老师傅手抖了，干不动了，手艺怎么办
- **行话**：清洗、加固、补全、做旧、全色、封护、颜料层、胎体、釉面、修复档案、最小干预原则
- **职业困境**：一辈子修复无数国宝，展览铭牌上从无名字；修复标准之争（修旧如旧 vs 修旧如新）撕裂同行；收入与投入严重不匹配，年轻人留不住；老修复师退休后手艺断代
- **感情线可能**：考古学家（田野与修复台的互补）、博物馆馆长之子（体制内外的碰撞）、艺术品鉴定师（真伪之间的博弈）、收藏家（文物商品化的对立面）
- **参考作品气质**：《我在故宫修文物》纪录片质感 + 都市职场婚恋的克制温度

### A2. 非遗传承人 / 守艺人

- **人性冲突**：传统消亡的见证者 vs 现代转型的挣扎者——她守的不是手艺，是一整代人的记忆
- **叙事质感**：快时代里的慢手艺，一针一线皆是光阴
- **核心场景**：
  1. 老街工作室里日复一日地重复同一道工序，窗外是喧嚣的商业街
  2. 年轻学徒来了又走，最后一个学徒离开时她没有挽留
  3. 直播带货的镜头前，她笨拙地展示手艺，弹幕只关心价格
  4. 非遗展演上，她看到隔壁展位用机器批量生产的"非遗产品"卖得更好
  5. 深夜独自做工，手上的老茧和器物上的纹理一样深
- **行话**：烧制、錾刻、缂丝、髹漆、打坯、传承人认定、非遗名录、工美大师、师承关系、原产地保护
- **职业困境**：手艺没人学——年轻人要吃饭；手艺养不活自己——订单少、价格压得低；被商业"利用"还是被时代"遗忘"的两难；非遗认证的体制化困境
- **感情线可能**：文化投资人（传统 vs 商业的核心碰撞）、独立设计师（传统纹样与现代审美的融合）、同门师兄妹（共同守护的羁绊）、回乡创业青年（带着新思路回来的人）
- **参考作品气质**：《百鸟朝凤》的坚守感 + 都市婚恋的现实温度

### A3. 乡村医生

- **人性冲突**：守护乡民的健康 vs 自己人生的出路——走不了，也不忍走
- **叙事质感**：山路上的药箱，被遗忘的医疗前线，一个村子的健康全压在她肩上
- **核心场景**：
  1. 深夜被敲门叫醒，背着出诊箱打着手电走山路去给老人看病
  2. 村卫生室里药品不全，她用有限的资源做最难的临床判断
  3. 留守儿童发烧，爷爷奶奶只信偏方，她苦口婆心劝去镇医院转诊
  4. 新农合报销政策年年变，她帮村民填表格、解释条款比自己看病还累
  5. 曾经的同窗在城里三甲医院升了副主任，她在村里连职称都评不上
- **行话**：听诊、输液、测血压、出诊箱、冷链管理、新农合、转诊、全科诊疗、基药目录、慢病随访
- **职业困境**：什么都得会（全科+儿科+急救），什么都缺（设备、药品、培训），村民信任偏方不信医生；没有编制、没有退休金、没有接班人；干了一辈子乡村医生，身份依然是"赤脚"的
- **感情线可能**：支教老师（两个"留守"者的相互温暖）、镇卫生院医生（上下级医疗体系的碰撞）、返乡创业青年（带着外面世界回来的人）、县医院外科医生（偶尔回乡义诊的人）
- **参考作品气质**：《山村女教师》的乡村质感 + 医疗线的生死重量

### A4. 户外救援向导

- **人性冲突**：救人的代价是拿自己的命赌——每一次出搜都可能是最后一次
- **叙事质感**：与自然搏斗，与时间赛跑，山脊线上的人性试炼场
- **核心场景**：
  1. 深山搜救迷路驴友，三天没合眼，靠卫星电话定位最后一个信号点
  2. 悬崖绳降救被困者，岩壁湿滑，绳索磨损到肉眼可见的危险
  3. 暴风雪中扎紧急庇护所，失温的驴友在她怀里发抖，她自己也快撑不住
  4. 被救者家属投诉"救援太慢"，她沉默地收拾装备
  5. 队友在搜救中受伤，她第一次在山里崩溃大哭
- **行话**：GPS定位、绳索系统、锚点、搜救扇区、失温分级、紧急庇护所、卫星电话、地形图判读、救援直升机协调、搜索半径
- **职业困境**：每次出搜都赌命，被救者不感恩甚至倒打一耙；救援经费不足，装备靠自己贴钱；全年无休，手机不能关机；社会认知模糊——不是导游，不是消防员，是民间志愿者
- **感情线可能**：消防员（体制内救援 vs 民间救援的碰撞）、被救者家属（从误解到理解）、山地救援志愿者（并肩出生入死）、气象预报员（天气预报是搜救的前哨）
- **参考作品气质**：《垂直极限》的紧张感 + 《烈酒玫瑰》的紧迫与柔情

### A5. 野生动物巡护员

- **人性冲突**：守护生灵 vs 面对盗猎者——她的枪口从未对准任何活物，但盗猎者的枪口对准过她
- **叙事质感**：原始丛林中的孤独巡逻者，星光照亮的不只是路，还有兽的脊背
- **核心场景**：
  1. 夜间巡护，红外相机里拍到雪豹的身影，她蹲在零下二十度的山口笑了
  2. 清晨发现盗猎遗留的钢丝套和血迹，沿痕迹追踪了一整天
  3. 与盗猎者正面遭遇，对方逃窜，她选择先救受伤的动物还是追人
  4. 保护区缓冲带，村民放牧与野生动物栖息的冲突调解
  5. 季度种群监测报告，数据让她看到某个物种在缓慢恢复
- **行话**：红外相机、足迹追踪、粪便样本、巡护路线、盗猎工具、栖息地、种群监测、缓冲带、核心区、反盗猎
- **职业困境**：与盗猎者正面冲突的生命危险；保护与发展的矛盾——村民要吃饭、动物要栖息地；长期离家驻守荒野，感情生活几乎为零；巡护员工资低、编制少，干得最苦的人最没保障
- **感情线可能**：动物研究者（学术与一线的碰撞）、保护区管理干部（政策与执行的拉扯）、调查记者（她想报道保护区的故事）、当地向导之女（山林里一起长大的人）
- **参考作品气质**：《可可西里》的苍凉 + 都市与荒野的碰撞张力

### A6. 投行分析师

- **人性冲突**：数字理性 vs 人性温度——她包装的公司上市了，她知道那个数字背后多少人失业
- **叙事质感**：凌晨四点的写字楼，亿万交易的冷静与疯狂，西装盔甲下的空心人
- **核心场景**：
  1. IPO路演现场，面对投资人提问，她把一个亏损公司的故事讲得天花乱坠
  2. 尽职调查期间深入工厂，看到裁员名单上全是干了二十年的老员工
  3. 凌晨四点在办公室改估值模型，第三十七版PPT，窗外是空无一人的CBD
  4. 谈判桌上，对面创始团队的眼神从信任变成敌意，她代表资方
  5. 项目庆功宴上，她喝了很多酒，去卫生间吐完回来继续微笑
- **行话**：DD（尽职调查）、估值模型、路演、簿记、定价、绿鞋机制、对赌协议、FA、PE/VC、招股书
- **职业困境**：帮不值得的公司包装上市的良心煎熬；行业性别天花板——同样级别女分析师要比男同事多付出三倍；35岁还没升VP就被淘汰；高薪但买不回健康和睡眠
- **感情线可能**：创业者（她尽调的对象，立场对立）、监管机构职员（合规与违规的边界）、同行竞争对手（亦敌亦友）、财经记者（她手里的内幕是他想要的线索）
- **参考作品气质**：《华尔街之狼》的金融张力 + 职场婚恋的利益纠葛

### A7. 危机谈判师

- **人性冲突**：理性共情 vs 情感消耗——她说对了每一句话救了人，但那些话也在掏空她自己
- **叙事质感**：一句话决定生死，沉默是最强的武器，电话线两端的灵魂博弈
- **核心场景**：
  1. 劫持现场，她拿着扩音器跟持刀者对话，三十米外狙击手已就位
  2. 天台边缘，想跳楼的少年说"你别过来"，她站住了，开始问他的名字
  3. 谈判复盘室，她逐字逐句回放录音，分析自己哪句话差点激怒对方
  4. 心理咨询室，她自己的督导问她"你上次哭是什么时候"，她答不上来
  5. 深夜接到被救者的电话，对方说"谢谢你救了我"，她挂了电话沉默了很久
- **行话**：积极倾听、情感标签、镜像技术、时间战术、核心诉求、退让阶梯、行为评估、触发点、降级策略
- **职业困境**：每次谈判都在跟操纵者博弈，自己的心理健康谁来保障；救人之后没有掌声——最好的结果就是"什么都没发生"；长期暴露在极端情绪中，亲密关系严重受损
- **感情线可能**：特警搭档（武力与话术的配合）、心理咨询师（她自己的疗愈者）、被救者家属（沉重的感恩）、重案组刑警（案件交叉合作）
- **参考作品气质**：《心灵猎人》的心理博弈 + 《起底观心术》的洞察质感

### A8. 茶艺师

- **人性冲突**：慢生活哲学 vs 快时代效率——她泡一壶茶要四十分钟，客人赶时间五分钟喝完走了
- **叙事质感**：在茶香中修行，在沸水里见人心，一杯茶的时间足够看清一个人
- **核心场景**：
  1. 茶室里她专注地温杯、投茶、注水，对面客人却在低头刷手机
  2. 春天上茶山收茶，她和茶农在雨里采了一整天鲜叶
  3. 评茶大赛上，她蒙眼品出了茶的产地、年份和工艺，全场安静了
  4. 老茶客去世，儿子来把她存的茶全卖了，她看着那些茶被搬走什么都没说
  5. 夜里独自喝一泡老茶，茶汤从第一泡到第十泡，味道从浓到淡像一生
- **行话**：杀青、揉捻、发酵、焙火、出汤、挂杯香、回甘、岩韵、毫香、茶气、醒茶
- **职业困境**：年轻人喝奶茶不喝茶，被当成"高级服务员"而非文化传承者；茶行业水太深——以次充好、炒作价格，她想守住底线但同行在砸锅；手艺人的收入撑不起体面的生活
- **感情线可能**：咖啡师（慢生活 vs 洋饮品的碰撞）、茶商（利益与品质的拉扯）、空间设计师（茶空间的审美共鸣）、回乡创业青年（在城市与乡村之间选择的人）
- **参考作品气质**：日式治愈系的安静感 + 中国茶文化的厚重 + 都市婚恋的慢节奏

### A9. 空中管制员

- **人性冲突**：秒级决策 vs 生命重量——一个指令调度二十架飞机，一个口误就是空难
- **叙事质感**：雷达屏幕前的隐形守护者，她指挥着天空，却从没上过天
- **核心场景**：
  1. 塔台值班，同时指挥八架飞机进近，雷达屏幕上密密麻麻的光点
  2. 天气突变，雷雨云压过来，三架航班同时请求改航，她在十秒内做出航线分配
  3. 紧急情况——一架飞机报告引擎故障，她清空所有跑道优先着陆
  4. 深夜班结束走出塔台，天边泛白，她已经十二个小时没合眼
  5. 模拟机考核，她做了一个完美的故障处置，教官说"你知道上次有人犯这个错吗？两三百人没了"
- **行话**：航线、进近、塔台、放行许可、滑行道、起飞许可、复飞、备降、间隔标准、飞行计划
- **职业困境**：不能犯一个错——一个错就是空难；高压下的心理崩溃是职业病；倒班生活摧毁感情——白天睡觉晚上上班，和伴侣几乎没有共处时间；薪资与责任严重不匹配
- **感情线可能**：飞行员（空地协同的默契与距离）、气象预报员（天气是她的变量也是他的日常）、地勤工程师（机场里的另一个孤独者）、异地恋人（聚少离多的代价）
- **参考作品气质**：《机长先生，雷雨请绕飞》的航空质感 + 塔台视角的隐形英雄

### A10. 天体物理学家

- **人性冲突**：仰望星空 vs 脚踏实地——她研究的是百亿光年外的星辰，工资要还三十年的房贷
- **叙事质感**：在宇宙的尺度里，人类的爱恨微不足道——但恰恰因此珍贵
- **核心场景**：
  1. 天文台观测站，她守了一整夜，望远镜对准的目标是一颗可能存在液态水的系外行星
  2. 数据处理中心，从海量噪声中提取出微弱的脉冲星信号，她盯着屏幕哭了
  3. 国际学术会议上，她用英语做报告，台下一位导师级的前辈提出了致命质疑
  4. 射电望远镜阵列现场检修，她在海拔四千米的山顶缺氧头痛
  5. 论文被顶刊拒稿，审稿人只写了一句话："insufficient novelty"
- **行话**：红移、光谱分析、暗物质、脉冲星、系外行星、光年、望远镜阵列、信噪比、数据拟合、观测窗口
- **职业困境**：研究数年可能一无所获——天文是"没有结果也是结果"的学科；学术圈内卷严重，非升即走；研究对象离地球几十亿光年，但房租要月付；社会认知是"研究这个有什么用"
- **感情线可能**：航天工程师（她看星星，他造飞船）、科学记者（他想写她的故事，她不擅长被打扰）、同门师兄（学术与感情的双重张力）、隔壁实验室的程序员（写代码处理她的数据）
- **参考作品气质**：《星际穿越》的浪漫感 + 学术圈职场的真实 + 婚恋的烟火气

### A11. 飞行员 / 民航机长

- **人性冲突**：天空与地面的距离——她在三万英尺做生死决策，地面上的人等她回家吃饭
- **叙事质感**：三万英尺上的孤独决策者，云层之上的冷静与云层之下的牵挂
- **核心场景**：
  1. 雷雨季节绕飞，雷达图上红色回波密布，她在驾驶舱做出航线判断
  2. 模拟机训练，引擎失效科目，她在七秒内完成应急处置，教官点头
  3. 备降后住进陌生城市的酒店，窗外是陌生的夜景，她给家里打了个电话没人接
  4. 航前准备，她仔细检查每一个仪表和系统，副驾说"机长你每次都这么仔细"
  5. 飞越雷雨后，乘客鼓掌，她只是握紧了操纵杆，想到女儿今天过生日她又缺席了
- **行话**：巡航高度、备降、航路天气、应答机编码、V1/VR决断速度、复飞、MEL（最低设备清单）、航前准备、绕飞、颠簸
- **职业困境**：聚少离多是常态——一个月在家不到十天；随时待命的神经紧绷；一个决策影响几百条命的精神负荷；身体负荷（高空辐射、时差、不规律饮食）；女性机长在行业中的隐性歧视
- **感情线可能**：空中管制员（空地协同的默契）、乘务长（同在机舱的伙伴）、地面工程师（她信任的机务）、异地恋人（距离是职业也是爱情的隐喻）
- **参考作品气质**：《机长先生，雷雨请绕飞》《共你揽晴空》的航空质感 + 聚少离多的感情拉扯

### A12. 人工智能研发工程师

- **人性冲突**：创造未来 vs 被未来淘汰——她写的代码可能在某一天让包括自己在内的所有人失业
- **叙事质感**：在代码和算法中寻找人类的位置，算力洪流中的造浪者也是溺水者
- **核心场景**：
  1. 实验室深夜，模型训练到第72小时，loss曲线终于开始收敛，她盯着屏幕不敢眨眼
  2. 产品上线前夜，发现模型在特定场景下有严重偏见，她按下暂停键，老板说"上线时间不能改"
  3. 论文截稿前48小时，她发现竞争对手已经发表了类似工作，三个月的心血变成"related work"
  4. 伦理审查会议上，她指出产品存在隐私风险，被市场部反驳"用户不在乎"
  5. 实习生问她"姐你怕不怕AI取代人类"，她笑了笑没有回答
- **行话**：预训练、微调、对齐、推理、token、参数量、梯度、蒸馏、RLHF（人类反馈强化学习）、agent
- **职业困境**：自己的工作可能让自己失业；技术迭代焦虑——半年前学的已经过时；AI伦理边界的良心拷问；行业高薪但35岁焦虑同样存在；女性在技术团队中的边缘化
- **感情线可能**：产品经理（技术与需求的永恒对立）、算法同行（同频共振也同频内卷）、传统行业从业者（两个世界的碰撞）、AI伦理研究者（她造工具，他研究工具的边界）
- **参考作品气质**：科技感与焦虑感并存 + 职场婚恋的时代命题

## 方向二：百业多姿百态

> 职业是镜子，照出人性。读者因为"共鸣"而追读。
> 核心要求：常见职业写出不常见的人性深度。
> 参考作品气质基准：《冬有暖阳夏有糖》《山村女教师》《共你揽晴空》《孤行者》《纹心尖》《在八月盛夏》《大医之路》

---

### B1. 急诊科医生

- **人性冲突**：生死抉择 vs 职业倦怠——她救得了别人的命，却救不回自己对这行的热情
- **叙事质感**：快节奏、高强度、刀尖上跳舞，凌晨三点的抢救室是人间修罗场
- **核心场景**：
  1. 抢救室同时送来两个车祸重伤者，她一秒钟判断谁先救
  2. 凌晨三点心脏骤停患者，按压到第20分钟，汗从额头滴到口罩上
  3. 家属在抢救室外质问"为什么先救别人"，她没有时间解释
  4. 连续值班36小时后蹲在楼梯间吃泡面，手机里是男友的未接来电
  5. 一个年轻患者没救回来，她洗手时发现自己的手在抖
- **行话**：心电监护、气管插管、除颤、CPR、分诊、绿色通道、DNR（不实施心肺复苏）、血气分析、中心静脉置管
- **职业困境**：救不救得了？该不该救？谁来决定？日复一日面对死亡后的情感隔离——她开始对死亡麻木，这比死亡本身更可怕
- **感情线可能**：师徒（带教老师，传承与冲突）、同事（同一个战壕的战友）、消防员（急救接力）、患者家属（注意职业边界）
- **参考作品气质**：《大医之路》《白色巨塔》的医疗质感 + 婚恋的情感拉扯

---

### B2. 产科医生

- **人性冲突**：新生的喜悦 vs 产房的残酷——她迎接了无数新生命，自己的婚姻却难产
- **叙事质感**：希望与绝望交织，产房门里是生的欢呼，门外是崩溃的家属
- **核心场景**：
  1. 剖腹产手术，胎心下降到60，她三十秒内切开子宫取出婴儿
  2. 高危妊娠夜班，妊娠高血压产妇突发子痫，她一边控制抽搐一边护胎儿
  3. 保大还是保小的家属谈话，丈夫跪在走廊里做不出选择
  4. 产妇大出血，她按压子宫两小时，血渗透了手术衣
  5. 产后抑郁的产妇复查，抱着孩子对她说"我不想当妈妈"
- **行话**：胎心监护、宫口扩张、产程、剖宫产、子痫、胎盘早剥、产后出血、羊水栓塞、新生儿评分
- **职业困境**：生育权的伦理困境——胎儿畸形要不要告知；产妇死亡时的自责与纠纷；"保大保小"这个伪命题对医生的折磨；长期接生导致对"生命奇迹"的感受力消退
- **感情线可能**：儿科医生（新生儿科与产科的接力）、麻醉师（手术台旁的默契）、产妇家属（从对立到理解的转变）、生殖科医生（生育两端的人）
- **参考作品气质**：《大医之路》的医疗质感 + 生育话题的社会厚度

---

### B3. 儿科医生

- **人性冲突**：治愈孩子的成就感 vs 面对孩子死亡的无力感——她治好了99个，第100个让她崩溃
- **叙事质感**：柔软与坚韧并存，口袋里永远有贴纸的白大褂
- **核心场景**：
  1. 儿科门诊，哭闹的孩子趴在她肩上不走了，她一边安抚一边写病历
  2. 罕见病诊断，她翻了一夜文献，确诊时家长哭了说"谢谢你相信我们"
  3. 夜班收了一个高热惊厥的婴儿，她在抢救的同时安抚歇斯底里的母亲
  4. 白血病病房，她给化疗完的小女孩扎辫子，女孩说"阿姨你的手好凉"
  5. 一个孩子没救回来，她在值班室哭了二十分钟，擦干眼泪去看下一个病人
- **行话**：发热惊厥、血常规、骨穿、腰穿、抗生素分级、雾化、生长发育评估、新生儿黄疸、脱水程度
- **职业困境**：孩子不能准确表达症状，诊断全靠经验和观察；家长情绪管理消耗大量精力；误诊压力——孩子的病变化太快；面对儿童死亡的创伤后应激
- **感情线可能**：护士长（并肩作战的默契）、幼儿园老师（懂孩子的人）、单亲家长（看病过程中产生的羁绊）、儿童心理咨询师（跨科室合作）
- **参考作品气质**：《大医之路》的医疗质感 + 儿童视角的温柔与残酷

---

### B4. 心理咨询师

- **人性冲突**：治愈他人 vs 自救困境——她帮别人理清内心的结，自己的结越系越紧
- **叙事质感**：安静的房间，暗流涌动的内心，每句话都是药也是刀
- **核心场景**：
  1. 咨询室里，来访者突然崩溃大哭，她递纸巾的手保持稳定，内心也在震颤
  2. 来访者表达了对她的依恋（移情），她必须温和而坚定地维护边界
  3. 督导会议上，她汇报案例时被指出"你跟这个来访者走得太近了"
  4. 深夜接到来访者自杀危机电话，她在电话这头一秒一秒地拖住对方
  5. 自己失恋后坐在咨询室里，发现"咨询师"的身份让她不敢找人倾诉
- **行话**：移情、反移情、阻抗、共情、设置、脱落、CBT（认知行为疗法）、精神分析、咨询联盟、危机干预
- **职业困境**：能不能跟来访者有感情？职业边界与人性温度的永恒拉扯；自己的心理问题谁来管——咨询师的咨询师也叫督导；长期吸收负能量的替代性创伤；社会误解——"你就是聊聊天还收钱"
- **感情线可能**：来访者家属（边界之外的羁绊）、同行督导（专业上的镜映与碰撞）、精神科医生（心理治疗与药物治疗的交叉）、来访者转变的关系（禁忌与挣扎）
- **参考作品气质**：《起底观心术》的心理质感 + 婚恋中的边界困境

---

### B5. 法医

- **人性冲突**：为死者言 vs 为生者难——她替死者说出真相，活人却不愿意听
- **叙事质感**：冷寂的解剖室，沉默的真相，手术刀下没有谎言
- **核心场景**：
  1. 解剖室，她从受害者的胃内容物推断出死亡时间，推翻了嫌疑人的不在场证明
  2. 案发现场，她蹲在地上三个小时检查尸体表面损伤，膝盖跪麻了
  3. 出庭作证，辩护律师质疑她的鉴定结论，她用数据一项项驳回
  4. 标本室整理旧案档案，一桩十年前的悬案让她发现了新线索
  5. 相亲对象知道她的职业后脸色变了，她笑了笑说"下次见面大概在太平间"
- **行话**：尸检、死亡时间推断、毒物分析、创口特征、窒息征象、尸斑、尸僵、腐败程度、硅藻检验、DNA比对
- **职业困境**：与死亡为伴，如何面对生者；社会偏见——"你做什么工作的"是最大的社交障碍；长期接触恶性案件的心理创伤；法医人手不足，一个法医要覆盖几个辖区
- **感情线可能**：刑警（工作搭档，生者与死者的搭档关系）、律师（法庭上的交叉）、殡葬师（同样与死亡打交道的理解者）、外科医生（手术刀与解剖刀的不同命运）
- **参考作品气质**：《法医秦明》的专业质感 + 孤行者式的职业孤独

---

### B6. 刑辩律师

- **人性冲突**：程序正义 vs 道德审判——她为明知有罪的人辩护，因为程序正义是所有人的盾牌
- **叙事质感**：法庭上的唇枪舌剑，暗处的道德煎熬，法袍之下是千疮百孔的良心
- **核心场景**：
  1. 看守所会见，当事人承认了罪行但对她说"你是我的律师你必须帮我"
  2. 法庭辩护，她指出证据链中的漏洞，旁听席传来被害人家属的哭声
  3. 深夜阅卷，在几千页的证据材料中找到关键的矛盾点
  4. 媒体围堵法庭门口，记者追问"你怎么能为那种人辩护"
  5. 败诉后当事人在庭上失控咆哮，她收拾文件平静地走了出去
- **行话**：辩护权、无罪推定、证据链、质证、非法证据排除、阅卷、会见、取保候审、认罪认罚、量刑建议
- **职业困境**：为明知有罪的人辩护的心理折磨；社会舆论压力——"你还有良心吗"；败诉的代价是当事人的自由甚至生命；刑辩律师的收入两极分化——大案肥案少数，法律援助案件杯水车薪
- **感情线可能**：检察官（法庭对立方，立场碰撞）、法官（裁判与辩护的微妙关系）、法律援助对象（从感恩到越界的可能）、被害人家属（理解与对立之间）
- **参考作品气质**：《legal high》的法律质感 + 道德困境的深度 + 婚恋纠葛

---

### B7. 检察官

- **人性冲突**：惩治犯罪 vs 个别正义——她追求的正义是制度性的，但站在面前的嫌疑人也是一个人
- **叙事质感**：正义的守护者，不完美的裁决者，每一个起诉决定都是一枚硬币
- **核心场景**：
  1. 审查起诉，证据不足但她直觉被告有罪，签还是不签起诉书
  2. 出庭公诉，对方律师是她的大学同学，法庭上两人寸步不让
  3. 侦查监督，她发现公安机关取证程序违法，顶住压力排除非法证据
  4. 被害人家属跪在检察院门口求她"一定要判死刑"，她蹲下来说"我尽力"
  5. 一桩冤案平反，她作为当年的公诉人站在媒体面前，一夜白了几根头发
- **行话**：审查起诉、提起公诉、侦查监督、抗诉、补充侦查、量刑建议、认罪认罚从宽、公益诉讼
- **职业困境**：证据不足怎么办——放走可能犯罪的人还是起诉可能无辜的人？被害人不谅解的情感压力；错诉的代价是当事人的人生；体制内的晋升压力与独立办案的矛盾
- **感情线可能**：刑辩律师（立场对立但彼此尊重）、警察（侦查与起诉的上下游）、被害人家属（沉重的信任）、法学院同窗（理想与现实的分化）
- **参考作品气质**：《孤行者》的正义坚守 + 婚恋中的立场碰撞

---

### B8. 家事律师（离婚 / 继承）

- **人性冲突**：法律理性 vs 情感撕裂——她在别人的离婚里看清了婚姻，自己的感情却一团乱麻
- **叙事质感**：在破碎的家庭中寻找秩序，每一份离婚协议都是一段关系的尸检报告
- **核心场景**：
  1. 离婚调解室，夫妻为争夺孩子的抚养权互相揭短，孩子坐在角落不说话
  2. 财产分割谈判，发现委托人隐瞒了财产，她要劝对方如实申报
  3. 家暴援助，她帮受暴妻子申请人身保护令，对方丈夫带人来律所闹事
  4. 继承纠纷，三兄妹为了一套房子对簿公堂，老母亲的遗像摆在法庭上
  5. 调解成功后委托人送来锦旗，第二天又打来电话说"他又打我了"
- **行话**：离婚冷静期、抚养权、探视权、财产分割、共同债务、人身保护令、遗产继承、代位继承、调解协议
- **职业困境**：看到太多婚姻的黑暗面，如何相信爱情和婚姻；当事人情绪失控的精力消耗；家事律师收费低、事务杂、情感劳动重；女性家事律师容易被当事人当成了情绪垃圾桶
- **感情线可能**：心理咨询师（她做法律，他做心理）、法官（调解席上的默契）、对方当事人律师（对手亦是知音）、正在经历离婚的客户（从专业关系到私人情感的边界问题）
- **参考作品气质**：《冬有暖阳夏有糖》的都市质感 + 婚姻撕裂中的温度

---

### B9. 调查记者

- **人性冲突**：追寻真相 vs 人身安全——她想让真相曝光，真相背后的人想让她闭嘴
- **叙事质感**：暗处的追光者，笔比镜头更锋利
- **核心场景**：
  1. 暗访黑工厂，她伪装成打工者进厂，手机藏在饭盒里录音
  2. 线人在深夜打来电话说"别查了，他们知道是你"，她挂了电话继续写稿
  3. 报道被总编压下，她在办公室跟主编拍了桌子，回报社的走廊灯忽明忽暗
  4. 报道发出后被起诉名誉侵权，她在律师陪同下出庭应诉
  5. 被报道对象找到她的住址，门口出现了不明来历的"礼物"
- **行话**：深度调查、暗访、线人、匿名信源、发稿、版面、总编、选题会、通稿、公关稿
- **职业困境**：真相公开后被报道者的命运——她毁掉了一些人的生活，即使是该被毁掉的；报道被压下的无力感；传统媒体式微，深度调查的生存空间越来越小；记者收入与付出严重不匹配
- **感情线可能**：同行（竞争与合作）、线人（从利用到信任的复杂关系）、刑警（真相路径不同但目标一致）、被报道对象的家属（意想不到的立场转变）
- **参考作品气质**：《聚焦》《孤行者》的记者质感 + 婚恋的现实温度

---

### B10. 刑警

- **人性冲突**：破案执念 vs 家庭瓦解——案子破了，家也散了
- **叙事质感**：与罪恶赛跑的人，深夜值班室的烟灰缸是她最忠实的伙伴
- **核心场景**：

- **职业困境**：技术再好也过不了学历门槛——晋升看学历不看手艺；社会认知偏差——"蓝领"的标签让她在家长会上不敢说自己做什么；制造业转型中的技能焦虑——老技术被淘汰，新技术来不及学
- **感情线可能**：工程师（她修他设计的机器）、车间主任（上下级关系的微妙）、同厂的技术工人（门当户对的踏实）、职校老师（她想学，他愿意教）
- **参考作品气质**：《冬有暖阳夏有糖》的市井质感 + 手艺人的尊严与爱情


---

### B22. 科研人员 / 基础研究

- **人性冲突**：求知的热忱 vs 现实的引力——她研究的基础科学没有直接应用价值，但人类需要有人仰望星空
- **叙事质感**：实验室是修道院，培养基是她的经文，显微镜下藏着宇宙的秘密
- **核心场景**：
  1. 实验室深夜，培养箱里的细胞终于长好了，她对着显微镜看了四十分钟不愿离开
  2. 课题申请被拒，评审意见说"创新性不足"，她在办公室坐了一下午没说话
  3. 带学生做实验，学生的操作不规范导致整批数据报废，她没发火只是重新来了一遍
  4. 组会汇报，导师指出她三个月的实验设计有根本性缺陷
  5. 一篇论文被引用了100次，但她的工资还是刚过万，同学在互联网公司年薪百万
- **行话**：课题、基金、SCI、影响因子、同行评审、实验设计、对照组、文献综述、组会、博士后
- **职业困境**：基础研究出成果慢且不直接产生经济价值；学术圈"非升即走"的淘汰机制；科研经费申请的内卷——花在写本子上的时间比做实验多；女性科研人员在生育黄金期与出成果期的双重压力
- **感情线可能**：企业研发工程师（学术与产业的碰撞）、同实验室博士后（同一个课题组的朝夕相处）、科学编辑（她写论文他改论文）、高中同学（走了完全不同的路的人）
- **参考作品气质**：《在八月盛夏》的学术圈质感 + 婚恋的现实温度

---

### B23. 导游 / 领队

- **人性冲突**：让所有人开心 vs 自己没有假期——她的工作是别人的假期，别人的假期是她的工作日
- **叙事质感**：背包里的麦克风和药箱，在风景里永远做不了风景的一部分
- **核心场景**：
  1. 带团途中游客突发疾病，她在异乡的医院里跑上跑下挂号缴费
  2. 景点讲解她讲了第一千遍依然声情并茂，游客却在低头拍照不听
  3. 游客投诉酒店条件差要求换房，她微笑着协调，自己住的是员工宿舍
  4. 带出境团，一个游客走失了，她在异国街头找了三个小时
  5. 旺季连轴转了四十天，终于休息那天她哪儿都不想去，躺在酒店床上看天花板
- **行话**：带团、地接、领队、行程单、自费项目、购物点、脱团、签单、踩线、团队签证
- **职业困境**：导游收入不稳定——底薪极低靠购物佣金和自费项目；没有正常的节假日——别人放假她最忙；职业声望低——被等同于"导购"；带团的体力消耗极大，年龄大了干不动
- **感情线可能**：酒店前台（每次带团都住那家酒店的默契）、摄影爱好者游客（短暂的旅途交集）、同团领队（同行理解同行的苦）、景区工作人员（固定地点等她路过的人）
- **参考作品气质**：《在八月盛夏》的旅途质感 + 聚散之间的浪漫

### B24. 设计师 / 视觉设计师

- **人性冲突**：审美追求 vs 商业妥协——她做了十版方案，甲方选了最丑的那版说"就这个吧"
- **叙事质感**：像素和矢量线之间的完美主义者，屏幕的光照着她的黑眼圈
- **核心场景**：
  1. 提案会议上，甲方指着她的设计说"logo能不能再大一点"，她深呼吸保持微笑
  2. 深夜改稿，第十三版和第一版越来越像，她把头埋在臂弯里笑出了声
  3. 灵感枯竭时她去美术馆看展，看到一幅画突然有了方向，在手机备忘录里画草图
  4. 设计获奖了，发到朋友圈收获了五十个赞，甲方打电话来说"借你奖状用一下"
  5. 新人交来的设计稿她改了两个小时，最后还是自己重做了
- **行话**：提案、改稿、甲方、审稿、矢量、分辨率、色彩空间、字体授权、版式、视觉规范
- **职业困境**：审美的主观性——"美"没有标准但甲方有标准；设计软件迭代焦虑——新工具层出不穷；AI出图对设计师的冲击——客户说"AI三秒出图你为什么要三天"；自由职业者的收入不稳定
- **感情线可能**：甲方市场总监（从甲方乙方到心动）、程序员（感性与理性的碰撞）、插画师（同频的创作灵魂）、广告文案（她做图他写字，天作之合）
- **参考作品气质**：《纹心尖》的艺术质感 + 都市婚恋的审美碰撞

### B25. 画家 / 职业艺术家

- **人性冲突**：纯粹的艺术 vs 活着的现实——她画了一千幅画，卖出去的还不够付画室的租金
- **叙事质感**：画布上的颜色比她的人生更浓烈，松节油的气味是她的氧气
- **核心场景**：
  1. 画室里她对着一幅画了三个月的画发呆，觉得哪里不对但说不出来
  2. 画廊代理谈判，画廊主要抽成60%，她犹豫了但对方说"你不签有的是人签"
  3. 艺博会现场，她的画挂在角落，隔壁画家的一面墙卖了八幅
  4. 有人出三千块买她画了两个月的油画，她拒绝了，回家后看着银行卡余额发呆
  5. 美院同学聚会上，做设计的开了豪车，她坐地铁来的
- **行话**：创作、画廊代理、艺博会、个展、群展、画幅、媒介、油画棒、丙烯、裱框
- **职业困境**：纯艺术几乎没有稳定收入——画卖不出去就吃不上饭；艺术市场的残酷——画廊只推能卖的画家；"画什么"的存在主义焦虑——画了十年还在找自己的风格；社会认知是"艺术家=不务正业"
- **感情线可能**：画廊策展人（商业与艺术的拉扯）、艺术评论家（他写她的画，文字比画还懂她）、收藏家（他买她的画，也想要她的人）、美术老师（稳定与自由的碰撞）
- **参考作品气质**：《纹心尖》的艺术灵魂 + 婚恋中理想与现实的撕扯

## 使用指南

### 职业选择原则

1. **方向一优先选"冷门"**：读者因为好奇行业而翻开，越冷门越有信息差红利。
2. **方向二优先选"共鸣"**：常见职业写出不常见的人性深度，读者因为"我也干这行"而追读。
3. **跨方向组合**：方向一的女主 + 方向二的男主（或反之），职业碰撞产生最大张力。

### 感情线配对原则

| 碰撞类型 | 示例 | 核心张力 |
|---------|------|---------|
| 对立职业 | 刑辩律师 × 检察官 | 法庭对手，庭下心动 |
| 互补职业 | 急诊医生 × 消防员 | 生命接力，他救出来她救回来 |
| 同频职业 | 调查记者 × 刑警 | 真相路径不同但目标一致 |
| 跨界职业 | 非遗传承人 × 文化投资人 | 传统 vs 商业的核心碰撞 |
| 阶层碰撞 | 保姆 × 白领精英 | 同一栋楼里的平行世界 |

### 职业调研清单

每个职业开写前必须完成：

- [ ] 核心工作流程（一天的日程表）
- [ ] 3个以上真实从业者访谈或访谈资料
- [ ] 行业现状（收入水平、就业前景、性别比例）
- [ ] 职业资格考试 / 准入门槛
- [ ] 真实的职业争议事件（用于剧情参考）
- [ ] 至少一部可参考的纪录片或行业报道

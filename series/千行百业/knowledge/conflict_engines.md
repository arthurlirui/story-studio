# 冲突引擎：职场 × 婚恋双线模板

> 本文件为《千行百业》系列提供可复用的冲突模板。
> 核心原则：职场线和感情线相互推动，而非各自独立。

---

## 一、双线交织的总原则

```
职场线 ────────────────────────────────→ 职业成长
  │                                        │
  │ 职场困境 = 感情催化剂                   │
  │ 感情冲突 = 职业反思点                   │
  │                                        │
  ↓                                        ↓
感情线 ────────────────────────────────→ 关系成长
```

**关键法则**：
- 不能删除职场线只剩感情线 = 故事仍然成立。
- 不能删除感情线只剩职场线 = 故事也仍然成立。
- 但两条线叠加时，必须产生 1+1 > 2 的效果。

## 二、短篇冲突模板（8-12 章）

### 模板 A：职业困境 → 感情催化

```
第1章：职业高光/危机 → 女主在专业领域展现能力
第2章：职业困境浮现 → 一个棘手案件/患者/事件
第3章：男主登场 → 因职业原因产生交集
  ├─ 可能：男主是"对立方"（检察官 vs 律师）
  ├─ 可能：男主是"需求方"（患者家属 vs 医生）
  └─ 可能：男主是"同行"（不同媒体的记者）
第4章：第一次冲突/合作 → 职业立场不同，摩擦
第5章：职业危机升级 → 更大的困难出现
  ├─ 证据不足、上司施压、舆论危机
  └─ 同时：男主看到了女主专业的一面
第6章：感情线升温 → 因职业困境产生情感共鸣
  ├─ 他在她最无助时提供了专业上的帮助
  └─ 不是"我帮你解决"，而是"我理解你的困境"
第7章：职业高光 → 女主用专业能力解决核心危机
第8章：双线收束 → 职业 + 感情阶段性和解
```

### 模板 B：感情冲突 → 职业反思

```
第1章：女主职业状态 + 情感状态（看似平静，实则暗流）
第2章：男主出现 → 旧识/旧爱/新认识
第3章：感情冲突爆发 → 因职业原因产生裂痕
  ├─ 例：他不能接受她做殡葬师
  ├─ 例：她的工作时间挤压感情空间
  └─ 例：他请她放弃职业，她不答应
第4章：职业困境 → 同时工作上遇到大麻烦
第5章：感情拉扯 → 他在她最困难时选择离开/留下
第6章：职业反思 → 感情冲突让她重新思考职业的意义
  ├─ "我为什么坚持做这行？"
  └─ "如果连他都理解不了，这行还有什么意义？"
第7章：职业突破 → 在专业上找到答案
第8章：双线收束 → 感情要么走向和解，要么走向告别
```

## 三、长篇冲突模板（四卷）

### 第一卷：职业入门 × 感情萌芽

**冲突引擎**：
- 职业：新人入行，面对行业真实面貌的冲击。
- 感情：遇见男主，因职业原因产生交集。
- 冲突点：职业的理想 vs 现实；感情的最初吸引 vs 职业边界。

**必备节点**：
1. 女主选择/进入这个职业的原因。
2. 第一个职业震撼：发现行业不是想象的样子。
3. 与男主第一次因职业原因见面。
4. 第一次职业小危机 + 男主的第一个反应。

### 第二卷：职业动摇 × 感情冲突

**冲突引擎**：
- 职业：遭遇重大挫折，开始怀疑职业选择。
- 感情：感情深化，但职业立场不同产生裂痕。
- 冲突点：继续还是放弃？感情还是职业？

**必备节点**：
1. 重大职业挫折（败诉、患者死亡、报道被压下）。
2. 感情升温，但职业问题成为障碍。
3. 一个让她重新思考职业意义的触发事件。
4. 感情出现危机：他/她因为职业原因选择离开。

### 第三卷：职业突破 × 感情考验

**冲突引擎**：
- 职业：找到自己的方向，在专业上突破。
- 感情：感情面临最大考验，但也是最深共鸣的时刻。
- 冲突点：职业成功 vs 感情代价；个人成长 vs 关系重建。

**必备节点**：
1. 女主在职业上取得关键突破。
2. 感情线到达最低点或最高点。
3. 职业和感情的双重考验（必须同时处理）。
4. 做出选择：不是"放弃什么"，而是"如何同时拥有"。

### 第四卷：职业坚守 × 感情收获

**冲突引擎**：
- 职业：面对职业生涯的终极考验。
- 感情：感情走向成熟。
- 冲突点：不是"要不要继续"，而是"以什么姿态继续"。

**必备节点**：
1. 终极职业考验（被行业封杀/大案审判/生命威胁）。
2. 男主的最终选择：他如何证明他理解她的职业。
3. 女主找到职业与感情的平衡。
4. 结局：不是完美，但真实。

## 四、职业 × 职业碰撞模板

两个不同职业的人走到一起，职业本身制造冲突。

### 经典碰撞组合

| 职业 A | 职业 B | 核心冲突 | 感情张力 |
|--------|--------|---------|---------|
| 刑辩律师 | 检察官 | 法庭对手 | 庭上对峙，庭下心动 |
| 急诊医生 | 消防员 | 生命接力 | 他救出来，她救回来 |
| 调查记者 | 刑警 | 真相路径不同 | 她要曝光，他要保密 |
| 心理咨询师 | 来访者家属 | 职业边界 | 不能爱上，但已经爱上 |
| 建筑设计师 | 拆迁户后代 | 城市记忆 | 她设计了拆他家的方案 |
| 殡葬师 | 医生 | 生命两端 | 他救不了的，她来送 |
| 非遗传承人 | 文化投资人 | 传统 vs 商业 | 他要把她的手艺产业化 |
| 外卖骑手 | 白领精英 | 阶层鸿沟 | 她在 30 楼开会，他在楼下送餐 |

### 使用规则

1. 两个职业的核心冲突必须贯穿始终，不能只在开头出现。
2. 感情升温必须伴随职业冲突的化解。
3. 男主不能因为"爱上女主"就放弃自己的职业立场。

## 五、职业困境 → 感情升温的 5 个经典场景

### 1. "他懂我的专业"场景
- 女主在工作中遇到一个专业难题，周围人都不理解。
- 男主用一句话点出她困境的本质，因为他在他的领域里也经历过。
- 不是"我帮你解决"，而是"我懂你在经历什么"。

### 2. "他尊重我的职业"场景
- 有人在公开场合贬低女主的职业（殡葬师/外卖骑手/乡村教师）。
- 男主站出来维护她，不是因为她是他喜欢的人，而是因为"这个职业值得尊重"。
- 关键：他的维护基于对职业的理解，不是基于个人感情。

### 3. "他看到了我最狼狈的样子"场景
- 女主在工作中经历了最崩溃的时刻（手术失败/败诉/报道被压下）。
- 男主恰好出现，看到了她最不想被别人看到的一面。
- 他的反应不是"我来安慰你"，而是"我陪着你，你不用说话"。

### 4. "他信任我的专业判断"场景
- 女主的专业判断与其他人的意见相悖。
- 男主在关键时刻选择相信她，不是因为感情，而是因为他了解她的专业水平。
- 这个信任，成为她职业道路上最重要的支撑。

### 5. "我们并肩作战"场景
- 职业危机同时波及两人（如果他也是同行）或两人需要合作解决。
- 在合作中，两人展现了各自的专业能力，互补而非替代。
- 感情在并肩作战中自然升温，而非"英雄救美"。

## 六、禁止的冲突模式

1. ❌ 男主用钱/权帮女主解决职业问题。
2. ❌ 女主因为感情放弃职业追求。
3. ❌ 职业冲突被简化为"选工作还是选我"的二选一。
4. ❌ 男主不理解女主职业 → 突然理解 → 完全改变（转变太生硬）。
5. ❌ 所有职业对手都是脸谱化坏人，所有职业盟友都是纯粹的好人。

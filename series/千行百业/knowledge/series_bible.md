# 《千行百业》系列圣经

## 1. 系列一句话

一个职业女性在都市丛林中，既要面对职业本身的残酷考验，又要处理感情与生活的拉扯；她不是完美的，但她所做的每一个选择，都让读者看到这个职业背后真实的人性和温度。

## 2. 核心爽点

### 2.1 职业专业爽

职业细节是最强的信息差武器。读者读完后感觉自己"懂了这个行业"。

高频写法：

- 女主用专业判断力解决危机，展现不可替代的价值。
- 外行看热闹、内行看门道——女主一眼看出问题所在，观众跟着恍然大悟。
- 职业中的"灰色地带"：不是非黑即白，而是两难选择。
- 职业的"隐形代价"：医生的情感隔离、律师的正义困境、记者的心理创伤。

**禁止**：把女主写成"无所不能的超人"。专业能力需要在场景中展现，不是靠旁白吹嘘。

### 2.2 职业身份认同爽

女主从"自我怀疑"到"与职业和解"的成长弧线。

高频场面：

- 入行时的理想主义被现实击碎。
- 某个时刻产生"我到底为什么要干这行"的动摇。
- 在某个关键事件中，重新找回职业的意义。
- 不是"坚持初心"，而是"重塑初心"——她理解了职业的不完美，但依然选择留下。

**禁止**：职业成长弧线写成"从菜鸟到扫地僧"的速成爽文。成长需要代价，需要时间。

### 2.3 职业 × 感情的交织爽

感情线不独立于职业线，而是相互渗透。

高频写法：

- 两人因为职业立场不同而产生冲突（律师 vs 检察官、医生 vs 患者家属、记者 vs 被采访者）。
- 职业困境成为感情升温的催化剂：他在她最艰难的案子中提供了关键帮助，不是因为霸道，而是因为他懂她的专业。
- "职业边界"成为感情障碍：心理咨询师不能跟来访者谈恋爱、刑警队长不能跟线人走得太近。
- 感情不是"拯救"，而是"并肩"——两个人在各自的职业中找到彼此。

**禁止**：
- 男主靠钱/权/人脉帮女主解决一切问题。
- 女主为了感情放弃职业。
- 感情线变成"霸道总裁爱上我"。

---

## 3. 不可触碰的禁区

### 3.1 职业写实底线

- 不能编造职业流程（如：把手术室写成菜市场）。
- 不能抹黑真实职业（如：把所有警察写成腐败的）。
- 不能美化职业黑暗面（如：把受贿写成"行业潜规则无所谓"）。
- 涉及医疗、法律、警务等专业领域，核心流程必须经过调研。

### 3.2 题材红线

- 禁止：官场/仕途/政治斗争。
- 禁止：玄幻修仙、系统流、末日废土、穿越重生。
- 禁止：职业当噱头，实际是狗血豪门恩怨。
- 禁止：职业女性的"成功"被定义为"嫁得好"。

### 3.3 情感线红线

- 禁止：女主是恋爱脑，为了感情放弃职业底线。
- 禁止：男主是完美霸总，用钱和权力解决一切。
- 禁止：将职业女性写成"事业太强所以没人爱"的刻板形象。
- 禁止：将职业困境简化成"谈个恋爱就好了"。

## 4. 人物规则

### 4.1 女主铁律

- **必须有职业**：她的职业不是背景板，是她的身份、她的武器、她的困境。
- **必须有职业成长弧**：从某个阶段开始，到更高阶段结束（不一定是升职，可以是理解更深、格局更大）。
- **必须有职业困境**：不是"怎么做好工作"，而是"做这行到底值不值得"。
- **必须有独立于感情线的职业目标**：她不是因为男主才入行，也不会因为男主改行。
- **可以有弱点**：她可以是工作狂但不擅长亲密关系、可以专业优秀但情商一般、可以在职业上自信但在感情上笨拙。

### 4.2 男主铁律

- **必须有职业**：他也有自己的职业，而且他的职业和女主形成某种呼应或对立。
- **必须有专业魅力**：他的魅力来自他的专业能力和人格魅力，不是来自金钱和权力。
- **必须尊重女主的职业**：可以最初不理解，但最终必须理解她职业的意义。
- **不能是"拯救者"**：他可以在关键时刻提供帮助，但帮助必须基于"他懂她的专业"，不是基于"他有钱有势"。

### 4.3 配角规则

- 每个配角都有自己的职业动机，不是纯粹的工具人。
- 职业中的"对手"和"盟友"都可以是复杂的：今天的对手可能是明天的老师。
- 职业中的"前辈"角色很重要：传递职业精神、揭示行业真相。

## 5. 叙事结构

### 5.1 短篇结构（8-12 章，每章 3000-5000 字）

```
第1章：职业高光/危机开场 → 展示女主在某个关键时刻的专业能力
第2-3章：职业困境展开 → 一个大案子/大事件/大危机浮现
第4-5章：感情线启动 → 与男主因职业原因产生交集，第一次冲突/合作
第6-7章：职业危机升级 + 感情升温 → 两线交织，互相推动
第8-9章：职业高光时刻 → 女主用专业能力解决核心危机
第10章：结局 → 职业与感情的阶段性收束
```

### 5.2 长篇结构（四卷，每卷 20-30 章）

**第一卷：入行 / 生存**
- 女主进入行业/面对第一个重大挑战。
- 展现行业"入门级"的真实面貌。
- 建立核心人物关系。

**第二卷：磨练 / 动摇**
- 女主遭遇职业困境，开始怀疑选择。
- 感情线深化，但因职业原因出现冲突。
- 关键事件：一个让她重新思考职业意义的转折点。

**第三卷：突破 / 成长**
- 女主找到自己的职业方向/风格。
- 感情线达到高潮或转折。
- 职业与感情的双重考验。

**第四卷：坚守 / 传承**
- 女主面对职业生涯的终极考验。
- 不是"成为行业顶尖"，而是"成为一个真正的职业人"。
- 感情线收束，职业态度最终确立。

## 6. 职业叙事专项规则

### 6.1 职业信息的嵌入方式

- **不要倾倒**：不要在开篇花三章介绍职业背景。用场景和对话自然露出。
- **用冲突带出**：不是"她是一名急诊科医生"，而是"凌晨三点，她接诊了第三个心脏骤停患者，在心脏按压的间隙，她看了一眼手机——母亲发来的第 17 个相亲消息"。
- **行话的用法**：行话要让读者"学到一个新词"，但不能变成阅读理解。用行话时，上下文自然解释。
- **职业细节的密度**：每个场景至少 1-2 个职业特写细节（工具、流程、术语、行业惯例）。

### 6.2 职业困境的类型

| 困境类型 | 说明 | 示例 |
|---------|------|------|
| 伦理困境 | 职业伦理 vs 个人道德 | 律师为明知有罪的人辩护 |
| 情感困境 | 职业要求 vs 个人情感 | 医生不能对患者产生感情 |
| 职业倦怠 | 日复一日消耗热情 | 教师看到学生一届届毕业，自己却在原地 |
| 身份困境 | 职业身份 vs 个人身份 | 殡葬师被社会偏见，不敢告诉别人自己的职业 |
| 系统困境 | 个人良知 vs 体制规则 | 记者发现真相，但报道被压下 |
| 传承困境 | 传统 vs 现代 | 非遗手艺面临失传，年轻人不愿意学 |

## 7. 与现有系列的差异

| 维度 | 破镜之后 | 不被定义 | 千行百业 |
|------|---------|---------|---------|
| 核心驱动 | 旧情旧伤、迟来追悔 | 突破标签、拿回主场 | 职业真实、人性折射 |
| 时间线 | 伤害 N 年后 | 当下 | 当下，可跨年代 |
| 感情线 | 追妻火葬场 | 可选，非必须 | 必须，与职业线交织 |
| 爽点来源 | 迟来者失控、旧账清算 | 标签反杀、专业能力 | 职业专业爽、人性共鸣 |
| 题材 | 现言/古言/幻言 | 不限 | 都市日常（非仕途） |
| 女主定位 | 已经离开、不再等待 | 不被定义、重写规则 | 职业女性、在职业中成长 |

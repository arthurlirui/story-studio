# 《千行百业》系列工程

## 系列定位

> **千行百业，笔绘人间**

《千行百业》是一个以职业为切口、描绘当代都市众生相的系列创作工程。

核心命题：

> 每一个职业，都是一扇看世界的窗。  
> 透过这扇窗，我们看到的不只是专业与技能，  
> 更是一个人在事业与生活之间做出的每一次选择，  
> 以及在时代洪流中，普通人如何守住自己的位置和尊严。

## 系列核心理念

### 三大创作支柱

**1. 职业真实感**
- 每个故事嵌入足够真实的职业细节：行话、流程、困境、灰色地带。
- 不做职业说明书，而是用职业细节推动人物和情节。
- 读者看完一本，对这个职业有一个"内部视角"的理解。

**2. 人性折射面**
- 职业是镜子，照出人性：医生的无奈、律师的两难、记者的危险、殡葬师的孤独。
- 每一个职业都天然对应一种核心人性冲突：生死、真假、善恶、新旧、快慢。
- 职场不是背景板，是人物性格和价值观的试炼场。

**3. 婚恋与成长并重**
- 主分类：职场婚恋。感情线和事业线同等重要，相互交织。
- 拒绝"恋爱脑"：女主的事业选择影响感情走向，感情选择也影响事业轨迹。
- 拒绝"霸总拯救"：男主的魅力来自专业能力和人格，而非金钱和权力。

### 系列口号

> 她不是某某的妻子，她是她自己职业里的那个人。

## 题材范围

- **限定**：都市日常（非仕途）、职场婚恋主分类。
- **时代背景**：当代中国都市（北京、上海、广州、成都、杭州、重庆、西安、昆明等），可跨不同年代（八零年代、九零年代、零零年代、当代）。
- **职业谱系**：医生、律师、记者、警察、消防员、殡葬师、非遗手艺人、心理咨询师、民航飞行员、建筑师、教师、社工、外卖骑手、程序员、厨师、花艺师、茶艺师、文物修复师等。
- **禁止**：官场/仕途、玄幻修仙、古代穿越、系统流、末日废土。

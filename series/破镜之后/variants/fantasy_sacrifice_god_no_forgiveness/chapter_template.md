# 章节生成模板

## 输入

- 书名：
- 卷名：
- 章节号：
- 章节标题：
- 当前时间线：伤害 N 年后 / 回忆 / 真相复现
- POV：女主 / 旧爱 / 新男主 / 反派
- 本章情节功能：
- 本章情绪功能：
- 本章必须出现的信息：
- 本章不能提前暴露的信息：
- 结尾钩子：

## 写作要求

1. 开头 300 字内进入冲突或情绪张力。
2. 每章至少推动一项：关系变化、证据变化、权力变化、情绪变化。
3. 女主不能无意义被动挨打；即使暂时忍，也要有目的。
4. 旧爱追悔必须通过行动而非空话表现。
5. 章末必须有钩子，但钩子要服务主线。

## 章节结构

### 1. 开场钩子

- 用动作/场景进入，不要先解释设定。

### 2. 冲突升级

- 旧人误判、反派施压、现实危机、新证据出现。

### 3. 情绪刺点

- 旧伤被触发，但不要一次讲完。

### 4. 主角选择

- 女主做一个能体现边界或能力的选择。

### 5. 章末反转

- 身份/证据/孩子/旧物/新人物/公开场合反转。

## 输出格式

```markdown
# 第X章 章节名

正文……
```

## 质量检查

- [ ] 本章是否有明确冲突？
- [ ] 女主是否仍有主体性？
- [ ] 情绪是否落在具体动作/物件上？
- [ ] 是否避免一次性解释全部往事？
- [ ] 结尾是否让读者想点下一章？

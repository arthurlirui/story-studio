# 《破镜之后》系列工程

## 系列定位

《破镜之后》是一个面向女频长篇网文批量创作的系列宇宙/知识库工程。

核心命题：

> 故事不从相爱开始，而从“主角已经离开、已经受伤、已经不再等待”开始。  
> 多年后，旧人、旧账、旧情重新上门，主角必须决定：复合、原谅、清算，或者彻底向前。

## 题材范围

- 现言：豪门、都市、婚恋、职场、娱乐圈、带球跑、闪婚、家族偏心、久别重逢。
- 古言：侯府、宫廷、和离、弃妇、替嫁、为奴、出宫、旧臣、帝王迟来深情。
- 幻言：仙侠、妖族、契约、轮回、前世误会、血脉觉醒、神明迟悔。

## 统一起点

所有故事都从“伤害 N 年后”切入，而不是从伤害发生时切入。

常见起点：

- 离婚/和离 N 年后。
- 分手/退婚 N 年后。
- 断亲/被逐出家门 N 年后。
- 出宫/出狱/离京/逃离侯府 N 年后。
- 为奴/替身/被误解生涯结束 N 年后。
- 主角已经拥有新身份、新事业、新孩子、新势力、新靠山。

## 情感主线

可走两种大方向：

1. **破镜重圆**：旧人迟来深情，真正认错、偿还、改变，主角在完全掌握主动权后选择重新开始。
2. **绝不原谅**：旧人跪求原谅，但主角不回头，完成打脸、复仇、事业/权力/新感情线爽感。

中间可以混合：男主追妻火葬场，但主角最终不选他；或者旧人赎罪成功但不是靠哭，而是靠承担代价。

## 参考气质

参考书单方向：

- 破镜重圆：
  - 《第五年重逢，驰先生再度失控》
  - 《离婚半年才怀，贺总，孩子真不是你的》
  - 《离京五年，被绝嗣帝王夺上龙榻》
- 绝不原谅：
  - 《出狱后，我闪婚了植物人大佬》
  - 《出宫后的第五年》
  - 《为奴三年后，整个侯府跪求我原谅》

## 使用方式

新建一本同系列长篇：

```bash
cd /home/openclaw/.openclaw/workspace/story-studio
python3 series/破镜之后/tools/new_variant.py \
  --title "离婚第五年，前夫跪在雨里求我回头" \
  --mode modern \
  --ending no_forgiveness \
  --wound "误会背叛+偏心错爱" \
  --years 5
```

生成后会在：

```text
series/破镜之后/variants/<safe_title>/
```

建立本书独立资料夹，但仍引用本系列知识库。
